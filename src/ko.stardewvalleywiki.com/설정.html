시작 안내서 플레이어 설정 조작 모바일 조작 체력 기력 스킬 일주기

스타듀밸리의 설정에는 2가지가 있습니다:

새게임 생성 시, 새 캐릭터 생성 창에서 좌큭 하단에 있는 버튼으로 "고급 설정"을 할 수 있습니다.
인게임에서 열 수 있고 "옵션" 탭을 선택하면 됩니다.

고급 설정

고급 게임 옵션

  • 마을 회관 꾸러미: 일반/리믹스
마을 회관에 등장하는 꾸러미를 변경합니다.
  • 광산 보상: 일반/리믹스
광산에서 얻을 수 있는 보상을 변경합니다.
  • 1년차 완성 보장 (체크박스)
첫해를 마치기 전에 떠돌이 상인이 반드시 붉은 양배추 씨앗을 팔도록 합니다.
  • 마을에서 몬스터 등장 (체크박스) (주의: 마을이라고 적힌건 오역입니다. 농장에서 몬스터가 나오게 해주는 옵션입니다)
몬스터가 밤에 등장할 수 있게 합니다.
  • 매출 이익: 일반/75%/50%/25%
아이템 판매 시 가격을 조절합니다.

멀티플레이어 옵션

  • 시작하는 오두막 개수: 없음/1/2/3
시작하는 오두막 개수.
  • 오두막 배치: 가까이/따로
오두막의 배치 거리를 정할 수 있습니다.

다른 옵션

  • 랜덤 시드 (텍스트 박스)
게임 월드를 랜덤화하는데 사용되는 시드입니다.

인게임 설정

일반

  • 자동 달리기 — 플레이어가 자동 달리기를 할지 안할지 변경할 수 있습니다. The "Run" keybind can be held to perform the opposite. Default checked.
  • 초상화 표시 — Toggle the portrait visibility and the name placement when interacted with normally (excludes dialogues that have multi-choices). If shown, the portrait appears to the right side of the dialogue with the name beneath. Otherwise, the name precedes the message, that is, <Name>: <Message>. Default checked.
  • 상인 초상화 표시 — Toggle the portrait visibility when interacted with as a merchant. Visibility and placements are similar to "Show Portraits" option. Default checked.
  • 도구 타격 위치 항상 표시 — Toggle the visibility of the red-border that appears around tiles that will be affected by the tool. Default unchecked.
  • 이동 중에 도구 타격 위치 숨기기 — Toggle the visibility of the red-border while moving. If checked, the visibility depends on the "Always Show Tool Hit Location" option. Default checked.
  • 게임패드 배치 타일 표시 — Default checked, requires controller.
  • 게임 창 비활성화일 때 자동으로 일시정지 — Toggles whether the game will pause whenever the game window loses focus. All audio and specific animations (e.g., the Journal button) will continue to play. Default checked.
  • 게임 패드 스타일 메뉴 사용 — Cursor snaps to menus. Even without a controller plugged in, this will cause the cursor to snap to the top left of inventory when using the Shipping Box.

사운드

  • 음악 볼륨 — Slider for volume of music. Default 75%
  • 효과음 볼륨 — Slider for volume of sound effects. Default 100%
  • 주변음 볼륨 — Slider for volume of ambient sounds. Default 75%
  • 발자국 볼륨 — Slider for volume of player character footsteps. Default 90%
  • 대사 타이핑 사운드 — Toggle whether typing of dialogue will be heard. If checked, the volume is affected by "Sound Volume". Default checked.

그래픽

  • 창 모드 — Drop-down options are: Windowed; Fullscreen; Windowed Borderless. Default Windowed Borderless.
  • 해상도 — Sets the resolution of the fullscreen window. Drop-down options relies on supported display modes of the default graphics adapter. Inactive if not fullscreen.
  • 메뉴 배경 — Toggle whether the menu screen displays a static seasonal image. Default unchecked.
  • 툴바 잠금 — Toggle whether the toolbar moves to the top of the screen when the player is near the bottom edge of any map. If locked, the toolbar loses opacity.
  • 확대/축소 — Decrease or increase the zoom level in increments of 5%. Default 100%, minimum 75%, maximum 125%.
  • 확대/축소 버튼 — Toggle whether the zoom level buttons appear beneath the day/time display in the upper right of the screen. Default unchecked.
  • 조명 품질 — Available options are: Lowest; Low; Med.; High; Ultra. Default Med.
  • 눈 투명도 — Slider. Default 0%. Slide to the left to increase snow transparency.
  • 번쩍이는 효과 표시 — Toggle whether effects (such as lightning) are visible. Default checked.
  • 하드웨어 커서 사용 — Toggle whether the game uses a sprite for the cursor. The hardware cursor will always be an arrow, even if hovering over doors or NPCs. Default unchecked.

조작

Stardew Valley features Controller support with Rumble functionality.

  • 게임패드 럼블 — Default checked, inactive without controller.
  • 툴바 스크롤 방향 반대로 — Toggle. If checked, scrolling down picks the item to the left and scrolling up picks the item to the right. Default unchecked.
  • 기본 조작법으로 되돌리기 — Reset all keybinds to default values. Does not ask for confirmation.
  • 확인하기/실행하기 — Set the keybind for checking or doing some action such as: interacting with NPCs, Calendar, Help Wanted, doors, ladders and elevators; picking up an item from the bag slot; picking up forage and loot. Default Right-Click, X
  • 도구 사용 — Set the keybind for using tools, or picking up forage and loot. Default Left-Click, C
  • 메뉴 열기 — Set the keybind for opening and closing the Menu (always opens to the Inventory tab). Default Escape, E
  • 일지 열기 — Set the keybind for opening and closing the Journal. Default F
  • 지도 열기 — Set the keybind for opening and closing the Map. Default M
  • 위로 이동 — Set the keybind for moving the character up. Default W
  • 왼쪽으로 이동 — Set the keybind for moving the character left. Default A
  • 아래로 이동 — Set the keybind for moving the character down. Default S
  • 오른쪽으로 이동 — Set the keybind for moving the character right. Default D
  • 채팅창 — Set the keybind for toggling the chat box. Default Question, T
  • 달리기 — Set the keybind for running (if "Auto Run" unchecked) or walking (if "Auto Run" checked) while the key is held. Default LeftShift
  • 인벤토리 칸 #1 — Set the keybind for selecting the first slot in the inventory and toolbar. Default 1
  • 인벤토리 칸 #2 — Set the keybind for selecting the second slot in the inventory and toolbar. Default 2
  • 인벤토리 칸 #3 — Set the keybind for selecting the third slot in the inventory and toolbar. Default 3
  • 인벤토리 칸 #4 — Set the keybind for selecting the fourth slot in the inventory and toolbar. Default 4
  • 인벤토리 칸 #5 — Set the keybind for selecting the fifth slot in the inventory and toolbar. Default 5
  • 인벤토리 칸 #6 — Set the keybind for selecting the sixth slot in the inventory and toolbar. Default 6
  • 인벤토리 칸 #7 — Set the keybind for selecting the seventh slot in the inventory and toolbar. Default 7
  • 인벤토리 칸 #8 — Set the keybind for selecting the eighth slot in the inventory and toolbar. Default 8
  • 인벤토리 칸 #9 — Set the keybind for selecting the ninth slot in the inventory and toolbar. Default 9
  • 인벤토리 칸 #10 — Set the keybind for selecting the tenth slot in the inventory and toolbar. Default 0
  • 인벤토리 칸 #11 — Set the keybind for selecting the eleventh slot in the inventory and toolbar. Default Minus
  • 인벤토리 칸 #12 — Set the keybind for selecting the twelfth slot in the inventory and toolbar. Default Plus

스크린샷

This feature is currently unavailable on Xbox and mobile.

윈도우, 맥, 그리고 리눅스

Clicking the camera button takes a screenshot of the entire map the player's character is currently in. The default zoom is 25% of original size, but can be changed via plus/minus buttons to 50%, 75%, or 100%. The default save location is:

  • Windows: %appdata%\StardewValley
  • Mac and Linux: ~/.local/share/StardewValley/Screenshots

Clicking the screenshot button the first time creates a new folder called "Screenshots" and adds a button to open the folder directly from options.

You can also use this feature with the chat command /mapscreenshot. The command will let you specify filename and percentage as well, for example /mapscreenshot image01 100.

PS4와 스위치

Clicking the screenshot button takes a screenshot of the entire map the player's character is currently in. Screenshots are saved to the Album page on Switch and to Settings > Storage > System Storage > Capture Gallery on PS4.
