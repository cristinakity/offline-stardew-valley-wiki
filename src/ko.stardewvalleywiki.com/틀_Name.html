이 틀 대한 자세한 내용은 영어 설명서 페이지를 참조하십시오.

용도

이 틀은 관련된 페이지에 다음과 같이 입력하여 사용할 수 있습니다: 

{{Name}}

결과... [[]]

사용

이 템플릿은 일부 항목의 이름을 자동으로 한국어로 번역합니다. 자동으로 번역되지 않는 항목은 "alt"또는 "link"매개 변수를 사용해야합니다. 

{{name|Strawberry|link=한국어 항목 이름}}

예제

{{Name|Strawberry|link=딸기}}

결과 ... 딸기 

{{name|Rainbow Shell|link=무지개 껍데기}}

결과 ... 무지개 껍데기 

{{Name|Cranberries|2|link=크랜베리}}

또는 

{{Name|Cranberries|link=크랜베리|2}}

결과 ... 크랜베리 (2)
