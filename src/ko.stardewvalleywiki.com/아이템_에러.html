'아이템 에러는은 게임의 기본 데이터 파일에 있는 내부 ID 번호가 올바른 아이템 ID 번호와 일치하지 않아 생깁니다. ObjectInformation.xnb

아이템 에러는 가격, 체력, 기력, 설명 등 관련된 아이템 데이터가 없으므로 구매/판매이 {price|0}으로 출력되며 이와 함께 "?"라고 적힌 설명이 표시됩니다.

아이템 에러는 항목 ID의 위치와 일치하는 이미지를 스프라이트 시트에 표시하는데, 이 이미지는 항목이 아닌 다른 이미지(광석, 반짝임 효과 등)일 수 있습니다.

game version 1.5 이전에는 잘못된 ID로 아이템 ID 오류를 사용하여 아이템을 생성할 수 있었습니다.
