  • 질문 3: When you compare 칼리코 달걀 and en:Calico Egg, there is a description "A unique egg that can be found during the Desert Festival." in the eng wiki. How do I insert this or edit the infobox? I googled it for about an hour and couldn't find the answer. Also, this should be the official in-game description, isn't it? Is there any guideline if it can't be filled? Ipleproduction (토론) 2024년 6월 15일 (토) 04:38 (UTC)

The English wiki uses /doc format, while the other languages use "Englishdoc" instead. If you wish to add the info in en:Template:Bundle/doc to 틀:Bundle, append the info after {{Englishdoc}} and before </noinclude>. As an example, see 틀:Name.

To insert a description into an infobox, add the parameter |description = [translated description].

Alternatively, you can add the entry to 틀:Description.

Hope this helps! margotbean (토론) 2024년 6월 15일 (토) 10:31 (UTC)
