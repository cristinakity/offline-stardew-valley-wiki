Sell Price with Gemologist Profession

Since Diamonds are the best profitable Gem to put in Crystalariums, lads get confused to the Sell Price with the Gemologist Profession, it sells for 750g for Diamonds without the Profession, with the Gemologist Profession, it sells for 974g, on the Wiki it says 975g instead of 974g. This is incorrect to the Wiki. Refer to the game code for help troubleshooting. 9000 (talk) 8:39, 7 July 2023 (UTC)

Well, I would call this a game bug, since 750 * 1.3 = 975. Unfortunately, changing the game code might affect all prices in the game. So, I will put it on my "to do" list to add a parameter to the infobox to allow for price corrections. Please note that it will not happen overnight! It may take a while! But, thank you for pointing this out. margotbean (talk) 18:07, 7 July 2023 (UTC)
