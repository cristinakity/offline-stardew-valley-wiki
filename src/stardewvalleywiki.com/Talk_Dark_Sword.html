Restoring Health

Does this sword really have an 8% chance of restoring health when striking? Because I tested this in the game and never worked, unlike the Vampiric enchantment, which worked.

Could this be an error from update 1.5? Because I saw a video of news from update 1.4 and the cure worked. JaksStrange (talk) 15:35, 29 March 2021 (UTC)

I've been trying to find the 1.4 code that allowed it to heal, and not having any success. There may be a bug with the sword with v1.5, that the sword doesn't heal until enchanted. I don't know, and if that's true, is it intended or not? If you're sure that the sword never heals until it's enchanted, then you can make a bug report on the official forums. I'm sorry I don't have better answers at the moment! margotbean (talk) 16:29, 29 March 2021 (UTC)
I hoard all weapons so I justed tested it in the dangerous mines for 5 minutes and got multiple heal procs (~25 hp). I never enchanted it, but it has the "Vampiric" tag (purple star). For me it works, maybe bad luck? Dubesor (talk) 17:34, 29 March 2021 (UTC)
Oh, I see! The weapon comes with the enchantment when you first obtain it. That's definitely a change from v1.4 (that was NOT specifically in the release notes). I will attempt to update the page... --margotbean (talk) 17:47, 29 March 2021 (UTC)
the crit chance also got nerfed by 1% (from 5% to 4%), - thus it is now tied in Critchance with the Insect head (Info in Note) but I don't know which version because I don't have any files from pre 1.5 Dubesor (talk) 19:51, 29 March 2021 (UTC)
There were no changes in Weapons.xnb from v1.4 to v1.5 (for existing weapons), that's about all the help I can be on that topic. Either it changed pre-v1.4 or it was an error on the wiki. :/ margotbean (talk) 20:07, 29 March 2021 (UTC)
Update: The crit. chance for the Dark Sword changed in v1.4 -- it was lowered from .05 to .04. No one noticed, probably because it wasn't in the RELEASE NOTES. *ahem* Thank you all for your work in figuring this out, and for updating the wiki, I really appreciate it! margotbean (talk) 20:10, 29 March 2021 (UTC)

Old Sword Descriptions & Names

In the game code the Sword class seems to be old unused code, but it makes some references to item old names & descriptions in StringsFromCSFiles.xnb. These could be interesting for trivia sections.

The Dark, Holy, and Galaxy sword are referenced, as well as a sword named "Hero's Sword" which might've been either the Rusty Sword, Silver Saber or neither. QtheConqueror (talk) 21:00, 1 November 2023 (UTC)

You might notice that I reverted your edit to the page. Version 1.6 removes a lot of unused code from the game files, so the reference will soon be obsolete. The Dark Sword reference seemed to have come from a pre-release version of the game, as the old description is already listed on the page. No one can verify pre-release descriptions except ConcernedApe himself, so let's avoid making those. I think the reddit comment covers most of what you've said, and the rest is speculation. Feel free to contact CA and ask about them. margotbean (talk) 21:08, 1 November 2023 (UTC)
