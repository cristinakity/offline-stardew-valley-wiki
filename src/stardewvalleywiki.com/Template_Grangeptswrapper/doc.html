< Template:Grangeptswrapper

Description

This template creates a table header and footer for each category of items in the Grange Display at the Stardew Valley Fair. It is intended to be used in conjunction with Template:Grangeptsrow.

Parameters

Parameter Notes
First (unnamed) parameter header or footer
catname Name of category to display
note (Optional) Any notes about the items in the category
catnum Category number, can be found in Data/ObjectInformation.xnb
For categories containing multiple types of items, any category number from the group can be used (e.g., "Minerals" includes category -2 and -12; either one can be used).

Examples

{{Grangeptswrapper
|header
|catname = Animal Products
|catnum  = -5}}
{{Grangeptswrapper|footer}}

produces...

Animal Products 
Item Grange Points
Base Rancher
{{Grangeptswrapper
|header
|catname = Fish
|catnum  = -4
|note=''Note that [[Clam]] is not considered "fish", it is considered uncategorized (trash) when calculating score.''}}
{{Grangeptsrow|Blobfish|Blobfish|500|-4|qualities=0,1,2,4}}
{{Grangeptsrow|Lobster|Lobster|120|-4|qualities=0}}
{{Grangeptswrapper|footer}}

produces...

Fish
