Discussion on updates

Hello margotbean! I noticed that it has been a month since you updated Template:Collections Items Shipped, but this template has not been updated. I would like to ask if this means that there will be new fish added in the upcoming 1.6.9 version? Or is it just because you're busy with other work?

If it is the latter, then I can help update this template. I have a detailed understanding of the things to pay attention to when updating the template. It is not very difficult to translate this template into all 12 languages, as I only need to translate the names of the fish in the template according to the names in the game files.

What is your idea? Look forward to your reply! 第二个熺 (talk) 16:49, 28 June 2024 (UTC)

Hello! I have not updated the template because I am busy, yes!
If you would like to update the template in all 12 languages, that would be wonderful! You are correct, it is not difficult when you have the data files for all 12 languages. It is tedious, though.
You must add "Page Two" to the template in all 12 languages. Template:Collections Items Shipped contains the translations for "Page Two" in all languages already, so you can use those translations. You also have to add the page 2 image and extend the table cells to a second page. You can use Template:Collections Items Shipped as a guide for the spacing.
Please be sure to preview the hover links (blue boxes) before saving the changes!! They do not like to behave themselves, you have to yell at them (so to speak).
My best advice: Make the changes in your first language (Chinese?) first, preview them, save them in a data file on your computer, understand them, digest them, maybe even memorize them, and then make the changes in the other languages in data files on your computer. After that, preview and save the changes in one language, and proceed to the other languages. This was my method, and it worked for me, so I'm recommending it to you.
I have uploaded the new images, File:Fish Collection Page 1.png and File:Fish Collection Page 2.png already.
If you have any questions, please ask! Otherwise, good luck!! margotbean (talk) 17:22, 28 June 2024 (UTC)
