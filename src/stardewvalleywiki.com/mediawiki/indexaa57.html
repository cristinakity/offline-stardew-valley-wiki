Description

This template is used to provide a template that displays an infobox for tools.

Use

This template can be used by entering the following onto a relevant page. Do not use this on your user page. It is reserved for content pages. 

{{Infobox tool
|name         = 
|image        = 
|image2       =
|description  =
|previoustier = 
|nexttier     = 
|cost         = 
|improvements =
|skillreq     =
|soldby       =
}}

Parameters

{{Infobox tool
|name         = 
|image        = 
|image2       =
|description  = (optional -- populated by Template:Description if omitted)
|previoustier = Link to previous tier of tool (optional)
|nexttier     = Link to next tier of tool (optional)
|cost         = Cost in gold
|improvements = Improvements over the previous tier of tool (optional)
|skillreq     = Skill level required to unlock item (optional)
|soldby       = Link to merchant who sells the tool (optional)
}}
