Incorrect copyright year

A significant portion of copyrighted content was created after 2016 so this copyright notice is incorrect in a lot of cases. There's a few ways we can correct it:

  1. Add a template parameter so it shows the correct year for each copyrighted work. Everywhere we use the notice will have to be manually updated, seems like a lot of effort for very little gain.
  2. Add all valid copyright years, i.e. This file (or parts of it) is copyright © ConcernedApe 2016-2022, 2024.
  3. Just use a single year range, i.e. This file (or parts of it) is copyright © ConcernedApe 2016-2024. Not strictly speaking correct since there was no update in 2023 (afaik?), but looks neater.
  4. Remove the year. Might make the notice look bald.

Option 1 means identifying when each asset was published. Options 2 and 3 mean updating the notice up to once a year if Stardew Valley receives an update. Option 4 is set-and-forget. Thoughts? My hat stinks (talk) 20:53, 26 April 2024 (UTC)
