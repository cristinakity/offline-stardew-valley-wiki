The Treasure Totem is a crafted item that spawns up to 16 Artifact Spots in a circle surrounding the player when used on tillable ground. The circle spawns 3 spaces each to the north, east, south, and west, as well as 2 spaces each to the northeast, southeast, southwest, and northwest. If an obstacle blocks any of the 16 tiles, that artifact spot will not appear. The crafting recipe is obtained from claiming Foraging Mastery.

Bundles

The Treasure Totem is not used in any bundles.

Tailoring

A Treasure Totem can be used in the spool of the Sewing Machine to create the Totem Mask. It can be used in dyeing, serving as an orange dye at the dye pots, located in Emily's and Haley's house, 2 Willow Lane.
