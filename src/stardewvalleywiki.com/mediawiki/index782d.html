Template:Addlanglinks

From Stardew Valley Wiki
Jump to navigation Jump to search

Description

This template adds inter-language links to templates on the English-language wiki. In other languages, inter-language links are added via Template:Englishdoc (See, for example, German, Korean, Spanish.)

This template eliminates the need to add inter-language links manually to all English template documentation pages.

Specifically, this template adds the following code: 

[[de:Vorlage:{{PAGENAME}}]]
[[es:Plantilla:{{PAGENAME}}]]
[[fr:Modèle:{{PAGENAME}}]]
[[it:Template:{{PAGENAME}}]]
[[ja:テンプレート:{{PAGENAME}}]]
[[ko:틀:{{PAGENAME}}]]
[[hu:Sablon:{{PAGENAME}}]]
[[pt:Predefinição:{{PAGENAME}}]]
[[ru:Шаблон:{{PAGENAME}}]]
[[tr:Şablon:{{PAGENAME}}]]
[[zh:Template:{{PAGENAME}}]]
