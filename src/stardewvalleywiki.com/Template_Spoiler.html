Description

This template is used to mark pages or sections of pages that contain content released with v1.6 of Stardew Valley.

When v1.6 content is released on all platforms, this template will be removed.

Use

This template can be used by entering the following onto a relevant page. 

{{Spoiler}}

Example

{{Spoiler}}

Results in:

 Warning: Spoilers  

This page or section contains unmarked spoilers from update 1.6 of Stardew Valley. Players may want to avoid or be cautious toward reading this article/section.
