This page lists the productivity of processing fruits and mushrooms into Artisan Goods in a Dehydrator.

The products produced usually sell for more than the raw input items, so using a Dehydrator is almost always profitable. Items that lose gold are highlighted in red, and items whose net increase in value is zero are highlighted in orange. The profit made from processing items is lowest when using iridium quality and highest when using regular quality items.

This table has separate columns that reflect the effect of Tiller or Artisan professions, which reduce and increase the profit from a Dehydrator, respectively.

  • Increase in Value (g) for Base compares the selling price without any profession to the input item sold without any profession.
  • Increase in Value (g) for Tiller compares the selling price if processed is sold without Artisan but the input item is sold with Tiller.
  • Increase in Value(g) for Artisan compares the selling price if processed with Artisan and input item is sold with Tiller.

If any of these is negative (red), processing would yield a loss in value, so its recommended to either just sell the input item directly or process it in a different way.
