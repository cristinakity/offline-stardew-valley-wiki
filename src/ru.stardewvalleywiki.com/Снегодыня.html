Снегодыня — это фрукт, который вырастает из семян снегодыни за 7 дней. Это одна из пяти культур, которые могут дать гигантский урожай. Одну снегодыню можно купить за data-sort-value="2500">2500 з. на Пире Зимней звезды каждый год (шанс появления в магазине 25%).

Узелки

Снегодыня (5 ед.) может быть использована для наполнения Узелок Зимней звезды на доске объявлений (смешанной).

Рецепты

Снегодыня не используется в рецептах.

Пошив одежды

Снегодыня может быть использована в катушке Швейной машинки с Тканью чтобы создать Фермерские штаны Используется как белый краситель при Окрашивании одежды.

Задания

Снегодыня не используется в заданиях.
