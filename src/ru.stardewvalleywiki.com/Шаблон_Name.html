Применение

Этот шаблон можно использовать путем ввода следующего на соответствующей странице: 

{{Name}}

Результаты в ... [[]]

Описание

Этот шаблон используется для размещения предмета с его изображением. Изображение автоматически изменяется до ширины 24 пикселя, а название предмета становится ссылкой, если существует статья с таким же названием. Также шаблон добавляет третий параметр в скобках для обозначения количества, описания или других дополнительных деталей.

Для ссылки на жителей и других персонажей, используйте Шаблон:NPC.

Использование

Этот шаблон можно использовать, добавив следующий код на соответствующую страницу: 

{{Name|''Английское Название Предмета''|''количество''}}

Дополнительные параметры

Все дополнительные параметры, если они используются вместе с «количеством», должны быть указаны после значения количества!

Параметр Описание
size Размер отображаемого изображения (пример: size=48).
class Альтернативные классы. В данный момент доступен только inline, чтобы предотвратить перенос строки (пример: class=inline).
link Заменяет отображаемое название предмета и ссылается на указанную страницу. (пример: link=Кукуруза)
{{!}} Меняет слово и оставляет изображение предмета с ссылкой, полезно если нужно склонить название предмета, пишется только после параметра «link» (пример: link=Абрикос{{!}}Абрикосы)

Примеры

Порядок заполнения со всеми дополнительными параметрами: 

{{Name|Apricot|кол-во или описание|class=inline|size=48|link=Абрикос{{!}}Абрикосы}}

Результат:
Абрикосы (кол-во или описание)

При использовании параметры не идентифицируются по имени, а по порядку.

Пример использования этого шаблона: 

{{Name|Corn|30}}

Результат:
Кукуруза (30) 

{{Name|Preserves Jar|3}}

Кадка (3)

Примеры дополнительных параметров:

Параметр «size» задает размер отображаемого изображения: 

{{Name|Apple|size=48}}

Результат:
Яблоко 

{{Name|Apple|size=24}}

Результат:
Яблоко

Примечание: изменение размера вручную рекомендуется только в исключительных случаях.


Количество предметов или дополнительное описание пишется после названия предмета: 

{{Name|Melon|3}}

Результат:
Дыня (3) 

{{Name|Mining|Уровень 9}}.

Результат:
Горное дело (Уровень 9)


Параметр «class=inline» применяется для предотвращения переноса строки (элемент отображается на одной строке с другими, не занимая всю ширину): 

Требования к рецепту: {{Name|Foraging|Уровень 8|class=inline}}.

Результат:
Требования к рецепту: Собирательство (Уровень 8).


Параметр «link» заменяет отображаемое название предмета. (Если название предмета не переводится автоматически и создает красную ссылку, этот параметр позволяет указать название вручную.) 

{{Name|Corn|3}}

Результат:
Corn (3) 

{{Name|Corn|3|link=Кукуруза}}

Результат:
Кукуруза (3)


Параметр « {{!}} » заменяет слово и оставляет изображение предмета с ссылкой на первое, пишется только вместе с параметром «link». 

{{Name|Apple|link=Яблоко{{!}}Очень большое яблоко}}

Результат:
Очень большое яблоко
