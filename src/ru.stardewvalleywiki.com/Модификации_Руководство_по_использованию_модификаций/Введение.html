Вступление

«Мод» - это просто пакет файлов, который меняет Stardew Valley. Моды могут добавлять различные функции (например, показывать NPC на карте), изменять игровые механики (например, изгородь будет медленнее распадаться), вносить косметические изменения (например, делать ваш дом похожим на дом хоббита) и многое другое. Использовать моды легко! Для этого Вам просто понадобится...

  1. Установить SMAPI. Это загрузчик модов для Stardew Valley - он делает возможным запуск игры с поддержкой модов и позволяет модам напрямую взаимодействовать с игровым кодом. Он безопасно устанавливается вместе с Вашей игрой, и Вы можете удалить его в любое время.
  2. Распаковать моды в папку "Mods"

Ниже приведены частые вопросы по модам и помощь по началу работы.

Начало

Установить SMAPI

Для начала вам нужно установить загрузчик модов SMAPI. Нажмите на нужную ссылку для большей информации:

  • Инструкции для Android
  • Инструкции для Linux
  • Инструкции для Mac
  • Инструкции для Windows

Найти моды

Далее вы можете загрузить моды, которые хотите. Некоторые предложения:

  • самые популярные на Nexus;
  • рекомендации к первому прохождению;
  • можете поискать моды здесь:
    Сайт Описание
    Nexus Mods Большинство модов воссозданы здесь.
    (Регистрация аккаунта бесплатна.)
    ModDrop Здесь можно найти множество модов.
    Вы можете использовать аккаунт ModDrop (только для Windows), чтобы устанавливать/управлять/автообновлять ваши моды, или скачивать напрямую с сайта без дополнительной регистрации.
    Chucklefish mods Меньший источник модов, многие из них довольно стары (большинство все еще на Nexus или ModDrop).
    Stardew Valley forums Некоторые моды размещаются прямо на форумах. Некоторые скрытые драгоценности есть только здесь, но найти их может быть непросто.

Перед тем как скачать мод...

  • Работает ли он на последней версии игры? (SMAPI автоматически отключит несовместимые SMAPI моды, но вы сами по себе с XNB модами. Попытайтесь проверять описание мода.)

Найти папку с игрой

Затем вам нужно найти папку с игрой (ту, что содержит файл  Stardew Valley.exe). Вот, где её можно отыскать:

Платформа Путь
Windows GOG: C:\Program Files (x86)\GOG Galaxy\Games\Stardew Valley
Steam: C:\Program Files (x86)\Steam\steamapps\common\Stardew Valley
Linux GOG: ~/GOGGames/StardewValley/game
Steam: ~/.local/share/Steam/steamapps/common/Stardew Valley
Mac GOG: /Applications/Stardew Valley.app/Contents/MacOS
Steam: ~/Library/Application Support/Steam/steamapps/common/Stardew Valley/Contents/MacOS

На Mac, папка Content хранится отдельно:
GOG: /Applications/Stardew Valley.app/Resources/Content
Steam: ~/Library/Application Support/Steam/steamapps/common/Stardew Valley/Contents/Resources/Content

Всё ещё не нашли? Есть и другие способы:

more options 
Способ 2: Найти через SMAPI
  1. Запустите SMAPI.
  2. Введи эту команду в консоль SMAPI (второе, открывшееся окошко будет папкой с игрой): 
    show_game_files
Способ 3: Найти через Steam
  1. Зайдите в аккаунт в Steam.
  2. Нажмите правой кнопкой на Stardew Valley.
  3. Нажмите Properties.
  4. Нажмите Local Files tab.
  5. Нажмите кнопку Browse Local Files..., чтобы открыть окно с файлами.
Способ 4: Найти через GOG Galaxy
  1. Зайдите в аккаунт в GOG Galaxy.
  2. В боковой панели с играми нажмите правой кнопкой на Stardew Valley.
  3. Выберите Manage Installation > Show Folder, чтобы открыть папку с играми.

Установить мод

Чтобы установить мод, просто распакуйте файл .zip в папку Mods в папке с игрой. Убедитесь, чтобы каждый мод находился в отдельной папке. Например, если у вас установлен мод PineapplesEverywhere, структура файлов должна выглядеть так: 

Stardew Valley/
   Mods/
      PineapplesEverywhere/
         PineapplesEverywhere.dll
         manifest.json
Далее описывается контент из находящейся в разработке версии SMAPI 2.8, который может претерпеть изменения по мере приближения к релизу.
Если у вас есть много модов, вы можете собрать в отдельные подгруппы. SMAPI автоматически проверит каждую папку, пока не найдет файлы мода. 
Stardew Valley/
   Mods/
      Fruit mods/
         PineapplesEverywhere/
            PineapplesEverywhere.dll
            manifest.json
         ApplesNowhere/
            ApplesNowhere.dll
            manifest.json

Настройка модов

Некоторые моды имеют файл config.json в своей папке. Он может появится после того, как вы впервые запустили игру с этим модом. Этот файл позволяет настрайвать мод под свои усмотрения.

Чтобы редактировать файл config.json, просто откройте его в любом текстовом редакторе (Блокнот, Word). Удостоверьтесь, что ваша игра не запущена во время настройки, или изменения могут не проявиться. Можете посмотреть basic guide to JSON; проще говоря убедитесь, что все значения убраны в ковычки (например "value"). Вы также можете использовать это: JSON validator, чтобы убедиться, что формат верен.

Работа с модами

  • Чтобы обновить мод, скачайте его последнюю версию и перенесите новые файлы на место старых. Так вы замените старые файлы, но сохраните все настройки и данные созданные под вас.
  • Чтобы убрать мод, просто уберите его из папки Mods.

XNB моды

XNB моды заменяют содержимое папки Content вашей игры на собственные файла. Если мод содержит .xnb файлы и не имеет manifest.json, то это XNB мод. Это старая форма модов, которую не рекомендуется использовать, но вы можете изучить Модификации:Использование XNB модов, если хотите узнать больше.

Частые вопросы

Можно ли использовать моды на таких платформах, как: Linux, Mac, Windows, или на консолях?

Платформы, которые поддерживают работу с модами:

Платформа Поддержка модов
ПК Linux ✓ полная поддержка
Mac ✓ полная поддержка
Windows ✓ полная поддержка
Консоли Nintendo Switch ✖ не поддерживает моды
PS4 ✖ не поддерживает моды
XBox One ✖ не поддерживает моды
Мобильные Android ✓ поддержка большинства модов
iOS ✖ не поддерживает моды

Почти все моды будут работать на любой поддерживаемой платформе, поскольку SMAPI переписывает их для совместимости.

Могут ли моды испортить моё сохранение?

Да, но это крайне редко. Только несколько модов влияют на ваш файл сохранения вообще. Моды вряд ли повредят ваш файл сохранения. В большинстве случаев вы можете исправить повреждение сохранения, отменив последнее сохранение.

Если вы беспокоитесь, просто время от времени создавайте резервные копии файлов сохранения. Настоятельно рекомендуется, даже если вы не используете моды, так как повреждение сохранения почти всегда вызывается самой игрой. SMAPI также автоматически создает ежедневную резервную копию ваших сохранений; Вы можете получить их из папки Mods/SaveBackup.

Могу ли я позже прекратить использование модов?

Да, за некоторыми исключениями (смотри предыдущий вопрос), моды лишь частично затрагивают ваши сохраненные файлы, так что вы можете прекратить их использование в любое время.

Могут ли моды отключить достижения Стима?

Достижения Стима хорошо работают до тех пор, пока вы не запустите SMAPI через Стим. Будьте уверены, что следуете инструкции по установке для конфигурации Стима на Windows (изменения Стима не нужны для Linux или Mac). Если Стим работает на фоне игры, достижения тоже должны работать.

Могут ли моды работать в мультиплеере?

Да, версия Stardew Valley 1.3 всё ещё нова, так что проверьте SMAPI лист совместимостей чтобы увидеть, подходит ли мод к данной версии.

Как работают моды зависит от них самих. Зачастую каждый игрок ставит собственные моды, и вы можете смешать модифицированных и консервативных игроков. Некоторые моды не работают (или работают плохо) в мультиплеере, или могут влиять на других игроков. Попробуйте проверить описание мода в деталях мулитиплеера (часто под заголовком "совместимость").

Вот некоторые "приблизительные" правила для модов, к которым нет указаний:

type of mod effects summary
Патч контента Зависит от мода Стоит посмотреть в Multiplayer а именно в Content Patcher readme (применимо и для модов XNB).
Замена карты Работает локально Сюда входят моды, которые заменяют существующую карту (без добавления специальной логики с помощью SMAPI). Они видны только игрокам, которые их установили. Игроки без пользовательской карты увидят обычную карту и будут подчиняться нормальным границам (то есть они могут видеть, как другие игроки проходят сквозь стены, но не смогут пройти за ними).
Визуальные изменения Работает локально Это включает в себя кастомные портреты и спрайты, кастомный внешний вид фермера (кроме GetDressed или Kisekae), перекрашивание и т. д. Эти моды влияют только на игроков, которые их установили. Других игроков это не затронет. Если у обоих игроков одинаковые моды, они увидят одинаковые визуальные изменения.
Изменение времени Работает глобально Это включает в себя любой мод, который приостанавливает время, изменяет текущее время или скорость времени. Эти моды работают только при установке основным игроком (хостом), но влияют на всех игроков.
Кастомный UI Работает локально Сюда входят моды, которые добавляют новые иконки, всплывающие подсказки, меню. Эти моды влияют только на тех игроков, которые их установили. Других игроков это не затронет.
Изменение мира Работает глобально Сюда входят любые моды, которые изменяют вещи в мире, к примеру: прочные заборы, изменения характеристик монстров и т. д. Эти моды влияют на всех игроков, независимо от того, кто их устанавливает.
Другие моды Зависит от мода Подробности смотрите в описании мода или спросите у автора.

Что такое SMAPI?

SMAPI - это мод, который был загружен для Stardew Valley. Он прекрасно работает с GOG и Стимом, так же это совместимо с Linux/Mac/Windows, вы можете удалить его в любое время, и если вам понадобится помощь, есть дружелюбное сообщество. SMAPI требуется для большинства типов модов Stardew Valley.

SMAPI делает многое для тебя. К примеру...

  1. Загружает моды и прямо в игру предоставляет API(программный интерфейс приложения), которые тот может использовать.
  2. Переписывает моды для кроссплатформенной совместимости, благодаря чему вы можете использовать практически любой мод в Linux / Mac / Windows.
  3. Перехват ошибок. Если мод дает сбой или вызывает ошибку, SMAPI перехватит ошибку, покажет подробности ошибки в окне консоли и в большинстве случаев автоматически восстановит игру
  4. Обеспечивает проверку обновлений. SMAPI автоматически предупреждает вас, когда доступна новая версия мода.
  5. Обеспечивает проверку совместимости. SMAPI автоматически определяет, когда мод несовместим, и отключает его до того, как он вызовет проблемы.
