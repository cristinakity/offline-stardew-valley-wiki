Формат

Шляпы имеют жестко закодированную категорию -95 (Hats.cs::Load)

Индекс Поле Пример значения
0 Name
Название 		Trucker Hat
1 Description
Описание 		Mesh in the back to keep your head cool.
2 skipHairDraw
Пропустить изменение причёски 		false
3 ignoreHairstyleOffset
Игнорировать причёску при позиционировании шляпы 		true
4 Name (for language files other than English)
Наименование (для языковых файлов, отличных от английского) 		Бейсболка

Если skipHairDraw "true", то игра не изменит прическу персонажа, когда он наденет шляпу. Если установлено значение "false", прическа игрока может измениться, когда он наденет шляпу.

Если для параметра ignoreHairstyleOffset установлено значение "true", это означает, что игра будет позиционировать шляпу независимо от прически вашего персонажа. Например, для такой шляпы, как Бант-бабочка, вы захотите установить его в положение "false", чтобы бант регулировал положение, которое будет прикреплено к выбранной прическе вашего персонажа. Однако для чего-то вроде повязки на глазу она должна оставаться в одном и том же положении на голове вашего персонажа независимо от того, какие волосы вы выбрали, поэтому для нее должно быть установлено значение "true".
