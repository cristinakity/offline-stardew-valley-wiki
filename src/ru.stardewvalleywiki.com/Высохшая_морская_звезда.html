Высохшая морская звезда
Морская звезда из первозданного океана. На удивление хорошо сохранившийся экземпляр!
Информация
Места для артефактов: Пляж (10%)
Чудовища: нет
Другие источники: Древний клад Сокровища на рыбалке
Цена продажи: data-sort-value="40">40 з.

Высохшая морская звезда — это артефакт, который можно выкопать в месте для артефакта на пляже, выловить в сундуке с сокровищами или достать из древнего клада.

Музей

Высохшая морская звезда, отданная в музей, засчитывается в общее количество отданных туда предметов для наград и достижений.

Пошив одежды

Высохшая морская звезда может быть использован в катушке швейной машинки, чтобы создать Золотую рубашку.
