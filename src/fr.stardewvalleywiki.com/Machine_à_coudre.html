Machine à coudre
À utiliser pour fabriquer des vêtements depuis le confort de votre maison!
Information
Source

Emily (Quête spéciale)
Prix de vente Ne peut être vendu

La Machine à coudre est un équipement utilisé pour coudre ou teindre des vêtements. Le joueur la reçoit par lettre après avoir complété la quête spéciale d'Emily "Pierres précieuses rajeunissantes". Cet objet fonctionne de la même manière que la machine à coudre du 2 allée des Saules.

Si le joueur perd cet objet, il peut être racheté dans la Boutique d'objets perdus de la Forêt Secrète, pour data-sort-value="10000">10 000po.

Historique

  • 1.5 : Introduite.
  • 1.6.9 : Si le joueur perd cet objet, il peut être racheté dans la Boutique d'objets perdus de la Forêt Secrète.
