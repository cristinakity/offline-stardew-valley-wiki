Modèle:Name

  • Langue
  • Suivre
  • Modifier

Voir la page de documentation en anglais pour plus d'informations sur ce modèle.

Utiliser

Ce modèle peut être utilisé en entrant ce qui suit dans une page pertinente. 

{{Name}}

Produit ... [[]]

Paramètres

Ce modèle traduit automatiquement les noms des différents éléments en français. Les éléments qui ne sont pas automatiquement traduits doivent utiliser le paramètre link. 

{{Name|nom de l'élément en anglais|link=nom de l'élément en français}}

Exemples

{{Name|Strawberry|link=Fraise}}

Résulte en... Fraise 

{{Name|Rainbow Shell|link=Coquille arc-en-ciel}}

Résulte en... Coquille arc-en-ciel 

{{Name|Cranberries|2|link=Canneberges}}

ou 

{{Name|Cranberries|link=Canneberges|2}}

Résulte en... Canneberges (2)
