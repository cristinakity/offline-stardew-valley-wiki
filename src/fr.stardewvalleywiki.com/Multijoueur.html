Le Multijoueur est un mode de jeu optionnel dans lequel 1 à 8 joueurs peuvent rejoindre le même monde via un écran partagé, sur un réseau local ou par adresse IP. Le multijoueur est disponible sur les versions PC (Linux/Mac/Windows), console (PS4/Switch/Xbox One) et mobile (iOS/Android), mais pas sur PS Vita.

Le joueur qui a créé le monde est "l'hôte" et les autres joueurs sont des "fermiers". L'hôte a le fichier de sauvegarde sur son système et peut y jouer en mode multijoueur et en mode solo. Si l'hôte n'est pas en ligne ou si la ferme n'est pas actuellement ouverte aux autres joueurs, les fermiers ne peuvent pas accéder au monde ni à leurs personnages dans ce monde. La propriété de la ferme ne peut être transférée aux fermiers sans modding.

Restrictions

  • Sur console seulement:
    • Les consoles n'autorisent que 1 à 4 joueurs, même après la mise à jour 1.6.
    • Le multijoueur à distance nécessite un abonnement aux services en ligne de cette console. (Le multijoueur local en écran partagé est gratuit.)
    • Il n'est pas possible d'avoir à la fois des joueurs en écran partagé et des joueurs à distance.
    • Sur Switch seulement, le mode écran partagé est limité à deux joueurs. Chaque joueur a besoin d'une manette complète ou de deux Joy Cons.
  • Le multijoueur multiplateforme (par ex. PS4 ↔ PC) n'est pas supporté, excepté les versions entre PC (par ex. Windows ↔ MacOS).

Héberger un monde

Un monde peut être créé spécifiquement pour le multijoueur, ou converti d'un monde solo en multijoueur à tout moment.

Commencer un nouveau monde multijoueur

Pour commencer un nouveau monde spécifiquement pour le multijoueur :

  1. Depuis l'écran titre, choisir Coop au lieu de Nouveau.
  2. Cliquer sur l'onglet Héberger.
  3. Cliquer sur Héberger une nouvelle ferme.
  4. Soyez sûr d'avoir ajouté une cabane pour chaque joueur qui rejoindra votre ferme. En option, une marge bénéficiaire peut être réglée pour décider si les joueurs partagent un portefeuille commun ou on chacun leur propre portefeuille. Les marges bénéficiaires ne peuvent pas être modifiées après la création de la ferme sans modification de la sauvegarde, mais le partage du portefeuille peut être modifié et les cabanes peuvent être construites ou démolies dans le jeu.

Démarrer le mode écran partagé

  1. Créer ou charger une sauvegarde normalement.
  2. Construire une cabane pour chaque joueur qui rejoindra votre ferme.
  3. Connecter une manette à l'ordinateur/console pour chaque joueur supplémentaire. (Le clavier sur PC est toujours réservé au premier joueur, bien qu'il puisse également utiliser une manette.)
  4. Ouvrir le menu du jeu et choisir l'onglet Options.
  5. Faire défiler jusqu'à la section Multijoueur et choisir Démarrer une partie en co-op locale.
  6. Demander à chaque joueur supplémentaire d'appuyer sur Start sur la manette connectée.

Si l'hôte a chargé le monde via l'onglet "Coop" sur l'écran titre, il peut avoir à la fois des joueurs à distance et en écran partagé dans le même monde (sujet à des restrictions).

Inviter des joueurs dans un monde solo

  1. Construire une cabane pour chaque joueur qui rejoindra la ferme.
  2. Sauvegarder et retourner à l'écran titre.
  3. Ouvrir la sauvegarde dans l'onglet Coop au lieu de l'onglet Continuer pour permettre aux autres joueurs de rejoindre la ferme.

Se connecter

Pour l'hôte, voir Héberger un monde. Pour les autres joueurs :

Sur PC
  • Via Steam: les fermes multijoueurs hébergées par vos amis Steam apparaîtront sur l'écran co-op (Si elles n'apparaissent pas, essayez d'envoyer une invitation via Steam dans l'onglet options du menu de jeu).
  • Via GOG Galaxy ou Steam, un code d'invitation apparaît dans les options du jeu. Les autres joueurs peuvent rentrer ce code dans l'écran de connexion si il ont aussi GOG Galaxy ou Steam.
  • Via écran partagé (voir Démarrer le mode écran partagé).
  • Par adresse IP directe. Cette manière de jouer en réseau nécessitera un port de pare-feu ouvert ou un service de LAN virtuel comme Hamachi. (Par défaut, le serveur utilise le port UDP 24642.[1])
Sur PlayStation 4
  • Les joueurs peuvent rejoindre à distance via le PlayStation Plus (nécessite un abonnement).
  • Jusqu'à quatre joueurs peuvent jouer en mode écran partagé (voir démarrer le mode écran partagé).
Sur Switch
  • Les joueurs peuvent rejoindre à distance via le Nintendo Switch Online (nécessite un abonnement). Les fermes multijoueurs hébergées par vos amis apparîtront sur l'onglet "Rejoindre".
  • Les joueurs Switch à proximité peuvent rejoindre via la communication sans fil locale (ne nécessite pas d'abonnement).
  • Jusqu'à deux joueurs peuvent jouer en mode écran partagé (voir démarrer le mode écran partagé).
Sur Xbox One
  • Les joueurs peuvent rejoindre à distance via le Xbox Live Gold (nécessite un abonnement).
  • Jusqu'à quatre joueurs peuvent jouer en mode écran partagé (voir démarrer le mode écran partagé).

Caractéristiques

Le mode Multijoueur est principalement identique au mode 1 joueur. Voici quelques caractéristiques spécifiques au mode multijoueur :

Ferme en quatre parcelles

La Ferme en quatre parcelles

La Ferme en quatre parcelles a été conçue pour le mode multijoueur. Chaque parcelle de la carte comporte une petite zone avec les avantages de certaines des autres cartes de ferme, ainsi qu'un petit étang pour les joueurs dans les parcelles en plus de la parcelle avec le grand étang pour puiser de l'eau. La carte des quatre parcelles est également disponible pour le mode solo.

Cabanes

Pour le placement des cabanes, voir Emplacements de cabanes

La ferme peut accueillir jusqu'à 3 cabanes (jusqu'à 7 sur PC), à sélectionner lors de la création de la ferme. Pour chaque cabane construite, un joueur supplémentaire peut rejoindre la ferme. Si la ferme démarre avec moins de 3 cabanes, Robine peut en construire d'autres dans la ferme (pour un total de 3 cabanes) pour data-sort-value="100">100po chacune. La construction d'une cabane est instantanée, donc les cabanes supplémentaires peuvent être construites le premier jour.

Le joueur qui héberge la partie peut aussi démolir une cabane à n'importe quel moment. Démolir une cabane retire son occupant de la partie, et place le contenu de son inventaire dans un coffre a proximité de l'endroit où se trouvait sa cabane.

Une cabane peut être améliorée à la Scierie par le fermier qui l'occupe uniquement. La disposition de l'intérieur d'une cabane améliorée est identique à l'intérieur d'une maison de ferme améliorée (maison de l'hébergeur ou du joueur en mode 1 joueur). Comme pour la maison de ferme, l'empreinte au sol de la cabane ne change pas après les améliorations.

Pour que Robine puisse améliorer une cabane, le fermier qui l'occupe doit être présent sur la ferme, ne serait-ce qu'en dormant le nombre de nuits nécessaire.

Marge de Profit

Quand vous créez une nouvelle partie multijoueur, vous pouvez choisir une marge de profit de 25%, 50%, 75% ou 100% (par défaut). Ceci aide a mieux équilibrer le multijoueur en réduisant l'argent récolté en vendant des objets (incluant les récoltes, la cueillette, les minerais et la nourriture cuisinée) par rapport au nombre accru de joueurs. Par exemple, avec une marge de profit de 25%, vous vendrez votre Blé pour data-sort-value="6">6po au lieu de data-sort-value="25">25po.

Les prix d'achat au Magasin général de Pierre et les prix d'achat pour le gazon, le sucre, le blé et le riz au Marché Joja sont aussi ré-ajustés. Le prix des autres objets (incluant les objets de la Forge et de la Poissonnerie, mais aussi les bâtiments et les améliorations d'outils) ainsi que l'or reçu en récompense des quêtes ne sont pas affectés.

Cela rend l'or plus difficile à obtenir au début du jeu; rendant ainsi plus important l'amélioration de vos compétences, la fabrication d'objets (pour les revendre), et la réalisation de quêtes d'embauche. Par exemple, Willy continuera de vendre des casiers à crabes pour data-sort-value="1500">1 500po malgré votre marge de profit diminuée; miner du fer et fabriquer vos propres casiers à crabes est plus économique.

Mariage

Les joueurs peuvent épouser un PNJ, ce qui fonctionne de la même manière qu'en mode solo. Chaque PNJ ne peut être marié que par un joueur à la fois, et les enfants sont ajoutés à la maison du joueur marié.

Les joueurs peuvent épouser un autre joueur en lui donnant une bague de mariage. Les joueurs mariés reçoivent un fruit étoilé dans une boîte à cadeau violette à côté de leurs lits après le mariage (équivalent au fruit étoilé reçu d'un conjoint PNJ). Une maison peur accueillir des enfants si elle appartient à l'un des joueurs mariés, si la chambre des enfants a été améliorée, si elle n'a pas déjà deux enfants et si les deux joueurs dorment dans son lit; le jeu laissera au hasard un joueur proposer d'avoir des enfants à l'autre, et s'il est accepté, l'enfant sera ajouté à la maison dans laquelle les deux joueurs dorment (si éligible), sinon la maison de l'autre joueur (si éligible), sinon le domicile courant.

Gameplay

  • Certaines décisions sont faites en collaboration par tous les joueurs en choisissant des dialogues ou en rejoignant des zones :
    • dormir, en entrant dans un lit;
    • démarrer un festival, en entrant dans la zone (du festival);
    • que ce soit pour finir le Centre Communautaire ou acheter l'adhésion Joja ;
  • Tous les joueurs partagent la même ferme. Il ya possibilité de séparer l'argent de chaque joueur à la création de la ferme.
  • Chaque joueur possède sa propre barre d'énergie, ses propres compétences et ses propres relations avec les autres villageois.
  • Chaque joueur possède sa boîte aux lettres et son porche.
  • Chaque villageois ne peut être marié qu'avec un joueur à la fois. Si plusieurs joueurs désirent la même compagne (ou le même compagnon), le premier à faire sa demande gagnera sa main.
  • Les événements de 14 coeurs ne peuvent se déclencher qu'une seule fois par PNJ dans le jeu hébergé. Par exemple, si un joueur se marie avec un PNJ, complète l'événement des 14 coeurs, puis le divore et qu'un autre joueur se marie avec ce même PNJ, il ne pourra pas avoir son événement des 14 coeurs.
  • Interactions entre les joueurs:
    • Les joueurs peuvent se marier entre eux et avoir des enfants. Les joueurs peuvent se demander en mariage en utilisant une bague de mariage (à fabriquer). La recette de fabrication de cette bague peut être obtenue au chariot de voyage pour data-sort-value="500">500po, et permettra au joueur de fabriquer une bague de mariage grâce à 5 lingot d'iridium et 1 tesson prismatique.
    • Les joueurs peuvent danser entre eux lors de la danse des fleurs (aucune condition nécessaire).
    • Les joueurs peuvent voir les scores des autres dans certains festivals.
  • Les joueurs peuvent regagner de l'énergie en se tenant dans le lit, sans dormir.

Quêtes et progression

  • La progression des quêtes est individuelle, même si les joueurs peuvent s'aider à les accomplir.
  • La difficulté des quêtes est basée sur celle du joueur ayant le plus haut niveau.
  • La progression des Mines est globale, et chaque joueur peut ouvrir les coffres pour lui-même (utiliser l'ascenseur pour atteindre les niveaux nécessaires aux quêtes des Mines fonctionne pour valider ces quêtes).
  • La collection du Musée est globale, et chaque joueur peut obtenir ses propres récompenses.
  • La récompense pour compléter le paquet du panneau d'affichage est appliquée à tous les joueurs, mais les objets de récompense ne sont donnés qu'au joueur qui termine le paquet.
  • Compléter un ensemble de paquets du Centre Communautaire ou les adhésions Joja débloquent les récompenses finales pour tous les joueurs. Cependant, seul le joueur qui a complété un paquet peut obtenir la récompense du-dit paquet.
  • La réparation du pont de la plage est aussi globale.
  • Chaque outil de chaque joueur a besoin d'être amélioré individuellement, même si les outils peuvent être échangés gratuitement entre les joueurs selon leurs besoins. Cela permet de supprimer les problèmes d'indisponibilité d'un outil durant son amélioration.
  • Chaque joueur peut acheter et placer des bâtiments sur la ferme, mais seul l'hôte de la partie peut améliorer la maison et les cabanes ne peuvent êtres améliorées que par leurs propriétaires respectifs.
    • Les permissions pour déplacer des bâtiments peuvent être changées à l'aide d'une commande
  • Chaque joueur peut obtenir son propre ensemble de fruits étoilé, même si la plupart ne seront obtenus que lorsque tous les joueurs auront effectué les tâches nécessaires.
  • Les recettes pour la cuisine et la fabrication sont obtenues individuellement, même si les joueurs peuvent bien sûr s'entraider.

Argent

  • L'argent est partagé par tous les joueurs ou divisé individuellement.
    • Une option permet cependant de séparer l'argent par joueur à la création de la partie, ou peut être changé grâce au "Registre de la ville" dans le Manoir du maire.
    • Si la séparation individuelle d'argent est activée après le démarrage de la partie, la somme commune sera divisée de manière égale entre tous les joueurs dans la partie.
    • Pour éviter que l'argent d'une ferme précédemment solo ne soit redistribué, il faut activer la redistribution avant qu'un autre joueur ne se connecte. Ce dernier recevra le montant de départ de data-sort-value="500">500po.
    • Le "Registre de la ville" permet d'envoyer de l'argent entre les joueurs si le portfeuille n'est pas commun.
  • Pour chaque objet placé dans un bac d'expédition, le prix de vente est déterminé par le joueur connecté qui possède les professions permettant de le vendre le plus cher possible.

Expérience

Chaque joueur possède ses propres points d'expérience. Le joueur qui commet une action sera celui qui gagnera ces points d'expérience.

  • Agriculture
    • Le joueur qui ramasse une récolte gagne les points d'expérience, peu importe qui a planté ou a contribué à la pousse.
    • Le joueur qui interagit avec un animal gagne les points d'expérience.
  • Extraction Minière
    • Le joueur qui frappe une roche en dernier (qui la casse) gagne les points d'expérience. Ce n'est pas partagé.
    • Le joueur qui amorce une bombe gagne les points d'expérience des roches détruites.
  • Cueillette
    • Le joueur qui ramasse un objet de cueillette gagne les points d'expérience.
    • Le joueur qui frappe un arbre, souche ou bûche en dernier gagne les points d'expérience. Ce n'est pas partagé.
  • Pêche
    • Vous gagnez les points d'expérience pour les poissons que vous attrapez.
    • Le joueur qui vide un casier à crabes gagne les points d'expérience, peu importe qui l'a placé ou y a mis les appâts.
  • Combat
    • Le joueur qui a tué le monstre gagne les points d'expérience. Ce n'est pas partagé.

Chance

  • Les joueurs peuvent avoir un niveau de chance journalière différent.
  • La Chance est calculée en moyenne entre tous les joueurs pour les calculs.

Temps

D'une manière générale, pendant une journée, le temps ne s'arrête que lorsque les joueurs assistent à un festival ou lors de l'utilisation de la commande /pause.
Lors de l'hébergement d'un jeu multijoueur, le temps se comportera comme en mode solo jusqu'à ce que le premier fermier rejoigne la partie. Cela signifie que l'hôte peut simplement garder son inventaire ou son menu ouvert en attendant qu'un autre joueur le rejoigne.

Dans une partie multijoueur avec 2 joueurs ou plus, le temps s'arrêtera lorsque :

  • La journée est terminée et un joueur choisit une profession ou n'a pas fermé le rapport d'expédition.
  • Les joueurs assistent à un festival qui comprend un événement principal. (Le temps passera à la soirée une fois le festival terminé).
  • L'hôte a mis le jeu en pause en utilisant la commande /pause. (Voir ci-dessous)
  • Le temps atteint 02:00 et un joueur est toujours dans une cinématique. (Ils s'évanouiront une fois la cinématique terminée.)
  • Le temps atteint 02:00 et un joueur a toujours son inventaire ou un menu ouvert. (Ils s'évanouiront une fois qu'ils sont fermés.)
  • Écran partagé uniquement: Si tous les joueurs sont sur un menu ou une cinématique, le temps s'arrêtera.

En aucune autre circonstance le temps ne s'arrêtera, ne sautera vers l'avant ou ne se déplacera à un rythme différent de la normale.

En particulier, le temps s'écoulera :

  • Dans n'importe quel menu, inventaire ou dialogue.
  • Quand l'inventaire d'une boutique est ouvert. (Continuer le commerce reste possible même lorsque le PNJ s'est éloigné ou s'est endormi.)
  • Dans une cinématique. (Voir ci-dessus. Vous pouvez voir les autres joueurs tout en étant invisible pour eux. L'heure n'est pas affichée.)
  • En jouant à un jeu d'arcade.
  • En ferrant un poisson.

Certaines actions restent possibles pendant que le jeu est mis en pause manuellement à l'aide de la commande /pause:

  • Les joueurs déjà dans un menu de boutique peuvent acheter et vendre des objets.
  • Les joueurs dans leur inventaire/menu peuvent parcourir et déplacer des objets.

Tchat

Le tchat est une fonctionnalité qui permet de communiquer entre les joueurs directement via le jeu. Le jeu diffusera également des messages dans le tchat, par exemple si un joueur est attrapé en train de fouiller une poubelle, ou s'il a atteint un certain nombre de pas.

Il existe également plusieurs commandes qui peuvent être exécutées dans le chat pour modifier les paramètres, envoyer des messages directs aux autres joueurs, etc.

Commandes de triche

Le jeu inclut des commandes de triche et de debug qui sont d'ordinaire inaccessibles.

Elles peuvent être activées, pour une sauvegarde, de la manière suivante :

  1. Modifiez le fichier de sauvegarde.
  2. Trouvez l'élément <allowChatCheats> et changez la valeur à true.
  3. Rechargez la sauvegarde de la partie normalement.
  4. Entrez /debug <command> pour exécuter une commande de debug.

Les deux commandes additionnelles seront activées :

Commande Description
money Syntaxe : money <montant>

Vous permet de régler le montant d'or en votre possession au montant spécifié.

Exemples :

  • /money 500 définit votre or total à data-sort-value="500">500po.
debug Syntaxe : debug <commande>

Exécutez n'importe quelle commande de debug (EN) via le tchat.

Exemple :

  • /debug itemNamed pufferfish rajoute un Poisson-globe dans votre inventaire.

Limite du nombre de joueurs

La limite de nombre de joueurs par défaut est de 7 (8 joueurs au total avec l'hôte), mais cette limite peut être changée sur PC (sans avoir à recourir aux mods) en modifiant manuellement les fichiers de configuration du jeu.[2] Notez bien que cette manipulation est expérimentale et peut provoquer des instabilités.[3]

Pour changer la limite, il faut aller chercher ces fichiers :

  • Windows
    • %AppData%\StardewValley\startup_preferences
  • macOS ou Linux
    • ~/.config/StardewValley/startup_preferences

Changer la valeur de <playerLimit>-1</playerLimit> au nombre de joueurs voulus dans le jeu, par exemple <playerLimit>10</playerLimit> pour autoriser 10 joueurs.

La valeur de -1 utilise le nombre de joueurs par défaut.

Exploits

  • Si deux joueurs ou plus prennent un objet ou cassent une pierre en même temps, ils recevront tous l'objet correspondant, ce qui s'apparente à une duplication d'objet.
  • Si un joueur casse quelque chose qui a une chance de larguer un butin, comme un Nœud d'iridium, le jeu appliquera cette chance à tous les joueurs. Si un joueur qui n'a pas cassé l'objet réussi à cette chance, l'objet de butin apparaîtra à proximité.
