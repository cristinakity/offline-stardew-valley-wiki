Le Journal détrempé est un type de Déchets qui peut être attrapé lors de la pêche pendant toutes les saisons, dans n'importe quel endroit, en utilisant soit une Canne à pêche, soit un Casier à crabes. Il peut également être trouvé dans des Poubelles, et n'importe quel poisson de Casier à crabes peut produire du journal détrempé lorsqu'il est placé dans un étang à poissons.

Le Journal détrempé peut être recyclé en torches ou en Tissu.

Paquets

Le journal détrempé peut être recyclé en Tissu, qui est utilisé dans le   Paquet Artisan dans le cellier.

Recettes

Le journal détrempé n'est utilisé dans aucune recette.

Couture

Le journal détrempé peut être utilisé dans la bobine de la Machine à coudre pour créer la Chemise poubelle.  

Quêtes

Le journal détrempé n'est utile à aucune quêtes.
