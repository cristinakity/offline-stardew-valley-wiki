Machine à café
Prépare automatiquement une tasse fraîche à café chaque matin.
Information
Source Quête spéciale "Cadeaux pour George"
Prix de vente Ne peut être vendu

La Machine à café est un équipement qui prépare automatiquement une seule tasse de Café le matin. Elle est reçue par courrier après avoir terminé la quête spéciale d'Evelyn: "Cadeaux pour George".

Si le joueur perd cet objet, il peut être racheté dans la Boutique d'objets perdus de la Forêt Secrète, pour data-sort-value="10000">10 000po.

Historique

  • 1.5 : Introduite.
  • 1.6.9 : Si le joueur perd cet objet, il peut être racheté dans la Boutique d'objets perdus de la Forêt Secrète.
