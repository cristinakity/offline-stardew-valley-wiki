Le Slime pétrifié est un minéral qui peut être trouvé dans une Géode ou une Géode-omni. Il peut également être récupéré dans une Boule de Slimes produite dans un Clapier pour Slime.

Paquets

Le Slime pétrifié n'est utilisé dans aucun paquet.

Recettes

Le Slime pétrifié n'est utilisé dans aucune recette.

Couture

Le Slime pétrifié peut être utilisé dans la bobine de la machine à coudre pour créer un T-shirt de Slime.

Quêtes

  • 1 Slime pétrifié est demandé par les Slime-Jack pour augmenter la capacité d'accueil de leur Étang à poissons de 7 à 10.

Historique

  • 1.0 : Introduit.
  • 1.1 : Est un butin possible des Boules de Slimes.
  • 1.4 : Peut désormais être utilisé en Couture. Peut être demandé dans une quête d'Étang à poissons.
