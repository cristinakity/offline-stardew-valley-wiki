Jeux sauvegardés

  • Langue
  • Suivre
  • Modifier
Pour plus d'information sur les sauvegardes sur les appareils mobiles, voyez Jeux sauvegardés mobiles.

A la fin de chaque cycle de jour, une "sauvegarde" s'effectue, c'est à dire, lorsque le joueur va au lit, ou si il s'effondre de fatigue à 2:00. A noter que toute action réalisée pendant une journée est définitivement perdue si le joueur quitte le jeu sans terminer la journée. En multijoueur, le jeu sauvegarde après que l'intégralité des joueurs passent le menu résumant le gain d'argent avant de se réveiller le matin.

Les sauvegardes sont taguées avec la dernière version du jeu qui a été sauvegardée (par exemple <gameVersion>1.6</gameVersion>). Une version du jeu plus ancienne ne sera pas capable de charger un fichier de sauvegarde d'une version plus récente, mais un fichier d'une version plus récente peut charger une version plus ancienne (par exemple quand vous sauvegardez pour la première fois après une mise a jour du jeu), le fichier de sauvegarde sera tagué avec le numéro de la sauvegarde plus récente.

Un fichier de sauvegarde peut être transféré à un autre type de console (par exemple de mobile à PC) si la version du jeu sur l'appareil recevant n'est pas plus ancienne que celle du fichier de sauvegarde. Par exemple, un fichier de sauvegarde version 1.4 provenant de la Switch peut être transféré à un PC avec le jeu mis à jour à la version 1.6, mais un fichier de sauvegarde version 1.6 ne peut pas être ouvert par un appareil avec le jeu en version 1.4.

Format de sauvegarde

Les sauvegardes sont répertoriées en tant que fichiers XML sur votre ordinateur ou appareil mobile. Elles sont répertoriées séparément de votre jeu, signifiant que vous pouvez réinstaller ou mettre à jour le jeu en toute sécurité sans perdre vos sauvegardes. Sur PC, elles sont partagées entre différentes copies du jeu (ex., Si vous l'achetez sur GOG et sur Steam).

Chaque sauvegarde à un dossier tel que JonSnow_123456789, contenant deux fichiers principaux: JonSnow_123456789 and SaveGameInfo. Les deux fichiers sont nécessaires afin de charger la sauvegarde.

Trouvez vos sauvegardes

Attention: en multijoueur, le fichier de sauvegarde est seulement dans l'ordinateur du joueur principal.

Windows

  1. Tapez Windows + R pour ouvrir une boite de dialogue.
  2. Copiez-collez le text ci-dessous:
    platforme Chemin d'accès
    Steam ou GOG %appdata%\StardewValley\Saves
    Application Xbox %localappdata%\Packages\ConcernedApe.StardewValleyPC_0c8vynj4cqe4e\LocalCache\Roaming\StardewValley\Saves
    Note: Si vous possèdez également le jeu sur Steam ou GOG, les sauvegardes peuvent être stockées dans leurs dossiers à la place.
  3. Appuyez sur 'OK' pour ouvrir le dossier.

Linux

Le repertoire racine pour les fichiers de sauvegarde est spécifié par la variable d'environnement XDG_CONFIG_HOME, qui par défaut est ~/.config/

  1. Depuis Fichiers, cliquez Aller » Aller à un fichier.
  2. Entrez exactement ce texte: 
    ~/.config/StardewValley/Saves
  3. Validez pour ouvrir le dossier.

Mac

  1. Depuis Finder, cliquez Aller » Aller à un fichier.
  2. Entrez exactement ce texte: 
    ~/.config/StardewValley/Saves
  3. Validez pour ouvrir le fichier.

Mobile (Android)

  1. Ouvrez l'explorateur de fichier de votre appareil. Si votre appareil n'en possède pas, vous devrez en avoir un qui permet la navigation dans le stockage interne, par seulement pour rechercher un fichier spécifique.
  2. Sélectionnez le dossier StardewValley.
  3. Chaque sous-dossier à l'intérieur avec un nom dans le format JonSnow_123456789 est un emplacement de sauvegarde.

Mobile (iOS)

  1. Ouvrez l'application Fichiers.
  2. Sur iPhone, appuyez sur le bouton Recherche en bas à droite de votre écran. Sur iPad, appuyez sur le bouton en haut à gauche si vous avez besoin d'afficher le menu à gauche.
  3. Sous la section Emplacements, appuyez sur Sur mon iPhone/iPad.
  1. Sélectionnez le dossier StardewValley.
  2. Chaque sous-dossier à l'intérieur avec un nom dans le format JonSnow_123456789 est un emplacement de sauvegarde.
Sur iOS 12 ou précédent:

Les appareils encore sous iOS 12 ou précédent devront être connectés à un ordinateur Mac ou Windows pour avoir accès à leurs fichiers de sauvegarde.

  1. Sur Mac, Ouvrez le Finder. Sur Windows, téléchargez iTunes et ouvrez le. (Si votre appareil est sur macOS 10.14 ou précédent, ouvrez iTunes à la place de Finder.)
  2. Connectez votre appareil iOS via un câble USB.
  3. Dans Finder/iTunes, cliquez sur l'icone de votre appareil dans la barre du haut. (Si l'icone n'est pas affichée, essayez de déconnecter et reconnecter le câble USB.) La page de l'appareil devrait s'afficher.
  4. Cliquez sur Partage de ficihers dans le menu à gauche.
  5. Cliquez sur Stardew Valley dans la liste des applications qui est apparue.
  6. Le panneau Documents Stardew Valley montre quels fichiers de sauvegarde sont déjà sur votre appareil iOS. (Ils apparaissent comme des dossiers, mais vous ne pouvez pas naviguer dedans directement.

PS Vita

Attention: L'accès aux fichiers de sauvegarde Vita requiert un micrologiciel spécialisé (CFW).

  1. Naviguez jusqu'à ux0:/user/00/savedata/ dans le Shell Vita.
  2. Mettez en surbrillance le dossier PCSE01235 et appuyez sur triangle.
  3. Selectionnez "Ouverture décprytée" en haut de la liste.

Nintendo Switch

Attention: L'accès aux fichiers de sauvegarde de la switch requiert un micrologiciel spécialisé (CFW).

  1. Téléchargez et installez JKSV
  2. Ouvrez JKSV et naviguez à Stardew Valley.
  3. Sélectionnez 'Nouveau' et donnez lui un nom
  4. Dans soit l'outil d'accès à la carte SD Hekate ou un serveur FTP, copiez sd:/JKSV/Stardew Valley/NameOfTheSaveYouSaved.zip.

Transférer une sauvegarde

Backup ou transférer entre deux PCs

  1. Trouvez vos sauvegardes.
  2. Compressez votre dossier qui ressemble à JonSnow_123456789.
  3. Partagez ou sauvegardez ce fichier compressé.

Android

Android vers PC
  1. Connectez votre appareil à votre PC.
  2. Naviguez jusqu'à votre stockage interne partagé > StardewValley.
  3. Copiez le dossier de sauvegarde entier (celui qui ressemble à JonSnow_123456789) dans le dossier de sauvegardes de votre ordinateur.
  4. Retirez l'extension .bin de tous les fichiers du dossier transféré.
PC vers Android
  1. Si vous utilisez des mods, retirez tous les contenus avant de transférer vers mobile.
  2. Trouvez vos sauvegardes sur votre ordinateur.
  3. Connectez votre appareil à votre PC.
  4. Naviguez jusqu'à votre stockage interne partagé > StardewValley.
    • Le dossier n'apparaîtera pas si vous n'avez pas créé une sauvegarde sur Android avant cela.
  5. Copiez le dossier de sauvegarde entier (celui qui ressemble à JonSnow_123456789) dans le dossier de sauvegardes de votre appareil.

iOS

Note: Sur macOS 10.15 et supérieur cette fonctionnalité iTunes a été déplacée vers Finder

iOS vers PC
  1. Trouvez vos sauvegardes sur iTunes.
  2. Choisissez le dossier de sauvegarde dans iTunes et cliquez sur le bouton "sauvegarder".
  3. Cela va sauvegarder un dossier contenant deux fichiers (chacun requis pour que la sauvegarde fonctionne).
  4. Copiez le dossier complet vers Copy the whole folder to le dossier de sauvegardes de votre ordinateur.
PC vers iOS
  1. Si vous utilisez des mods, retirez tous les contenus avant de transférer vers mobile.
  2. Trouvez vos sauvegardes sur votre ordinateur.
  3. Trouvez vos sauvegardes sur iTunes.
  4. Copiez le dossier de sauvegarde complet (celui qui ressemble à JonSnow_123456789) dans le panneau 'Documents Stardew Valley' sur iTunes.
    Note: Il n'y a qu'un dossier pour chaque sauvegarde. Vous avez besoin de mettre chacun de ses dossiers dans iTunes, pour que chaque dossiers soient dans la 'racine' du panneau 'Documents Stardew Valley'. Vous ne pouvez pas mettre le dossier contenant toutes vos sauvegardes dans iTunes.
  5. Cliquez sur le bouton 'sync' d'iTunes pour synchroniser votre appareil iOS avec votre ordinateur. La sauvegarde devrait maintenant apparaitre dans le menu de chargement des sauvegardes du jeu, sur votre ordinateur.

PS Vita

Note: Le développement du jeu sur Vita a été abandonné, laissant pour la dernière version de Vita une équivalence avec celle PC version 1.2.33 (2017). C'est pourquoi, même si un fichier de sauvegarde Vita peut-être transféré sur PC/mobile, celui d'un PC/mobile moderne sera incompatible avec Vita.

Vita vers PC/Android
  1. Trouvez vos sauvegardes sur votre Vita.
  2. Appuyez sur triangle dans votre dossier de sauvegarde (celui qui ressemble à JonSnow_123456789) et selectionnez "Copier".
  3. Naviguez jusqu'à ux0:/data/.
  4. Appuyez sur triangle et selectionnez "Coller"
  5. Transferez le dossier de sauvegarde dans votre dossier de sauvegarde PC/Android via FTP ou USB avec le Shell Vita.
PC/Android vers Vita
  1. Si vous utilisez des mods, retirez tous les contenus avant de transférer vers mobile.
  2. Trouvez vos sauvegardes sur votre ordinateur (celui qui ressemble à JonSnow_123456789).
  3. Transférez votre dossier de sauvegarde dans ux0:/data/ via FTP ou USB avec le Shell Vita.
    • Android: Retirez l'extension .bin de tous les fichiers du dossier transféré en appuyant sur triangle et en selectionnant "Renommer".
  4. Appuyez sur triangle dans votre dossier de sauvegarde et selectionnez "Copier".
  5. Naviguez dans Navigate to votre dossier de sauvegarde Vita
  6. Appuyez sur triangle et selectionnez "Coller"

Nintendo Switch

Switch vers PC/Android
  1. Ouvrez JKSV et naviguez jusqu'à Stardew Valley.
  2. Selectionnez "Nouveau", et appelez le comme vous le souhaitez.
  3. Dans l'outil d'accès à la carte SD Hekate ou sur un serveur FTP, copiez sd:/JKSV/Stardew Valley/NomDuFichierSauvegardé.zip.
PC/Android vers Switch
  1. Placez le dossier de sauvegardes PC/Android dans un zip dans sd:/JKSV/Stardew Valley/.
  2. Ouvrez JKSV et naviguez jusqu'à Stardew Valley.
  3. Il est conseillé de créer une sauvegarde (autre part) avant de faire la prochaine étape.
  4. Appuyez sur Y sur la sauvegarde que vous venez de copier et maintenez A

Dépannage

Sauvegarde disparue ou ne chargeant pas

Voici comment retrouver votre sauvegarde dans la majorité des cas. (À noter que les problèmes de sauvegardes sont quasi majoritairement causées par le jeu en lui même, et non par les mods.)

  1. Trouvez vos sauvegardes.
    • Si les fichiers sont toujours présents, voici quelques manières de faire (essayez une à la fois):
      1. Si quelques fichiers ont _STARDEWVALLEYSAVETMP dans leur nom, retirez cela et relancez le jeu. Vous devriez avoir besoin de désactiver Steam Cloud pour Stardew Valley, car il pourrait continuer de renommer les fichiers à chaque fois que vous relancieriez le jeu. Pour cela, allez sur Stardew Valley dans votre librairie Steam, cliquez sur la route dentée et allez dans Propriétés, et décochez l'option pour synchroniser les sauvegardes cloud.
      2. Si vous avez recemment renommé ou créé un dossier (par exemple pour restorer une backup/sauvegarde), soyez sur que le nom du dossier est exactement pareil que le fichier qui ressemble à FarmName_0123456789.
      3. Essayez Try d'annuler la dernière sauvegarde.
      4. Si vous êtes sur PC, assurez vous que les fichiers du jeu n'ont pas comme une extension comme.bin. Si vous êtes sur Android, à l'inverse, vous pourriez avoir besoin d'ajouter l'extension .bin.
      5. Les sauvegardes modées pourraient ne pas être capable de se lancer sur le jeu de base, cela dépend des mods utilisés. Si vous avez précédemment joué avec des mods mais que vous essayez de lancer le jeu de base, essayez de reinstaller SMAPI et de jouer une journée. SMAPI devrait automatiquement nettoyer tout contenu customisé de votre sauvegarde. À noter que si vous avez toujours des objets customisés dans votre inventaire, ils se transformeront en items buggés et pourraient causés des bugs.
    • Si les fichiers sont manquants mais que vous avez SMAPI d'installé, voici comment retrouver une backup:
      1. Trouvez votre dossier du jeu.
      2. Ouvrez le sous dossier save-backups.
      3. Dézippez la dernière backup qui contient votre sauvegarde, et copiez le dossier de sauvegarde dans Saves.
  2. Si les corrections ci-dessus n'ont pas aidées, regardez ce guide (en anglais) ou demandez de l'aide; nous devrions être capable de récuperer votre sauvegarde pour vous.

Note: Ne gardez pas des dossiers de backups dans votre dossier de sauvegardes, car le jeu essayera de les lancer. Soit vous les déplacez dans un dossier différent, soit vous les supprimez.

Annuler la dernière sauvegarde

Si quelque chose s'est mal passé, vous pouvez annuler la dernière sauvegarde.

Pour une sauvegarde depuis Linux/Mac/Windows/Android:

  1. Trouvez vos sauvegardes.
    Vous devriez voir deux fichiers avec un suffixe _old dans le nom in the name; si ce n'est pas le cas, demandez de l'aide à la place !
  2. Faites une backup de votre dossier de sauvegarde juste au cas où.
  3. Supprimez SaveGameInfo et le fichier qui ressemble à NomDeLaFerme_123456789.
  4. Retirez le suffixe _old de SaveGameInfo_old et le fichier qui ressemble à NomDeLaFerme_123456789_old.

Pour une sauvegarde depuis iOS:

  1. Trouvez vos sauvegardes.
    Vous devriez voir deux fichiers avec un suffixe _old dans le nom in the name; si ce n'est pas le cas, demandez de l'aide à la place !
  2. Faites une backup de votre dossier de sauvegarde juste au cas où.
  3. Supprimez les fichiers qui ressemblent à VotreNom_123456789 and VotreNom_123456789_SVEMERG.
  4. Retirez le suffixe _old du fichier qui ressemble à VotreNom_123456789_SVBAK.

Avancé

Modifier une sauvegarde

 
Un ficiher de sauvegarde ouvert avec Sublime Text avec XML formaté.

Vous pouvez modifier votre sauvegarde pour corriger quelques bugs connus ou modifier certains de vos choix. Cela peut cassé de manière permanente votre sauvegarde si vous faites une erreur; n'oubliez pas de créer tout d'abord une backup. Vous devriez également éviter les outils d'éditions de sauvegarde automatique, car ils créent très souvent des bugs.

  1. Installez Sublime Text pour Linux, Mac, ou Windows. (Si vous avez déjà un autre éditeur de text qui supporte le formattage XML, libre à vous de l'utiliser à la place.)
  2. Installez l'extension Package Control pour Sublime Text.
  3. Utilisez l'extension Package Control pour installer les plugins Indent XML et XPath (voir pour comment ajouter des plugins).
  4. Créez une backup de votre fichier de sauvegarde juste au cas où quelque chose se passerait mal.
  5. Ouvrez votre fichier de sauvegarde (le fichier qui ressemble à Nom_0123456789) dans Sublime Text.
  6. Cliquez sur Selection > Format > Indent XML.
  7. Faites tous les changements que vous souhaitez et sauvegardez. (Voir les sections ci-dessous, ce guide pour changer l'host d'une sauvegarde co-op, et ce guide pour d'autre changements.)

Conseils:

  • Cliquez sur un élément, et appuyez sur CTRL + SHIFT + [ pour réduire toute la section qui le contient. C'est très souvent pratique quand vous souhaitez ignorer un grand champ de texte. Par exemple, en prenant ce texte dans le fichier de sauvegarde: 
    <ArrayOfInt>
    <int>1981</int>
    <int>0</int>
    <int>0</int>
    <int>0</int>
    <int>0</int>
    <int>0</int>
</ArrayOfInt>

    Cliquez sur l'un des champs <int> et appuyez sur ces touches, et cela se réduira en: 

    <ArrayOfInt> […]
</ArrayOfInt>
  • Cliquez sur n'importe quel élément, et regardez sur la barre de status pour voir le chemin d'accès vers où vous êtes dans le format de sauvegarde (comme /SaveGame/player/friendships/item[32]).

Dupliquer une sauvegarde

Vous pouvez dupliquer une sauvegarde à des fins de tests, ou pour explorer de nouvelles options sans avoir à tout recommencer.

Trouvez vos sauvegardes.

  1. Créez un nouveau dossier de sauvegarde:
    1. Dupliquez le dossier de sauvegarde cible.
    2. Renommez le nouveau dossier:
      • changez le nom comme ça vous pourrez le reconnaître dans le jeu;
      • changez le numéro à la fin par n'importe quel numéro qui n'est pas déjà utilisé par une autre sauvegarde.
    3. Ouvrez le nouveau dossier pour les prochaines étapes.
  2. Préparez les nouveaux fichiers:
    1. Supprimez (ou appliquez les prochaines étapes à, si vous voulez la backup également) tous les fichiers qui ont un nom se finissant par _old ou .back.*.
    2. Renommez le fichier qui ressemble à Nom_0123456789 pour qu'il ressemble au nom du nouveau dossier.
  3. Editez les données de sauvegarde:
    1. Ouvrez le fichier SaveGameInfo dans un éditeur de text comme Notepad.
    2. Regardez pour <name> ou <farmName> sur la première ligne. Votre nom de ferme devrait être juste après. Changez le en votre nouveau nom.
    3. Cherchez la deuxième occurence de votre nom (CTRL + F pour 'trouver') et changez le en votre nouveau nom.
  4. Editez votre fichier de sauvegarde:
    1. Ouvrez le fichier qui ressemble à Nom_0123456789 dans un editeur de texte comme Notepad.
    2. Regardez pour <name> sur la première ligne. Votre nom de ferme devrait être juste après. Changez le en votre nouveau nom.
    3. Cherchez la deuxième occurence de votre nom (CTRL + F pour 'trouver') et changez le en votre nouveau nom.
    4. Cherchez l'identifiant de votre ancienne sauvegarde (<uniqueIDForThisGame>, le nombre de votre ancien nom de fichier). Changez le en votre nouveau nombre.

C'est fini! Lancez votre jeu et regardez votre menu de chargement des parties; vous devriez voir les deux sauvegardes maintenant.

Changer le type de ferme

Si vous utilisez SMAPI (mods):

Vous pouvez changer votre type de ferme en chargeant votre sauvegarde dans le jeu, puis en enterant set_farm_type NUMERO_TYPE_FERME dans la fenêtre de console de SMAPI. (Vous pouvez entrer set_farm_type list pour voir la liste des numeros des types de fermes.)

Cela changera la carte de votre ferme, mais tout ce qui a été placé sur la ferme restera où cela a été placé. Vous pourriez avoir des choses à des positions invalides, comme des arbres dans des lacs, etc. Vous pouvez alors soit installer un Noclip Mode pour atteindre les objets qui sont trop loins, ou executer ces commandes dans la console:

commandes effets
world_clear farm debris retire les pierres, branches, herbes, et plants morts.
world_clear farm trees retire les arbres non-fruitiers.
world_clear farm fruit-trees retire les arbres fruitiers.
world_clear farm grass retire les grandes herbes.
world_clear farm everything retire tout, incluant les objets placés.
Si vous jouez sans mods:
Vous pouvez changez votre type de ferme en modifiant votre fichier de sauvegarde:
  1. Ouvrez votre fichier de sauvegarde pour le modifier.
  2. Cherchez <whichFarm>. Le nombre après cette balise ouverte est le numéro du type de ferme.
  3. Changez le nombre par celui que vous voulez:
    Code Type de ferme
    0 Ferme Standard
    1 Ferme de rivière
    2 Ferme forestière
    3 Ferme des collines
    4 Ferme en région sauvage
    5 Ferme en quatre parcelles
    6 Ferme de plage
    7 Ferme des champs des près

Cela changera la carte de votre ferme, mais tout ce qui a été placé sur la ferme restera où cela a été placé. Vous pourriez avoir des choses à des positions invalides, comme des arbres dans des lacs, etc. Vous pouvez prendre note de ce qui est inateignable, revenir à l'ancienne ferme et les déplacer/détruire, avant de revenir à celle que vous souhaitiez.

Supprimer un PNJ

Vous pouvez supprimer un PNJ de votre fichier de sauvegarde (par exemple pour réparer un PNJ bloqué de manière permanente dans un endroit invalide). S'il existe toujours dans les données du jeu, alors le PNJ réapparaitra quand vous rechargerez le fichier de sauvegarde. Cela doit être utilisé en dernier recours, et est inutile si vous utilisez des versions de SMAPI supérieures à la 3.0, car SMAPI essayera de supprimer ces données pour vous.

  1. Ouvrez votre fichier de sauvegarde pour le modifier. Assurez-vous de formater le fichier de sorte à ce qu'il soit lisible pour les instructions suivantes.
  2. Retirer le PNJ:
    1. Cherchez le nom de base du PNJ entre les différentes balises 'name' (par exemple pour supprimer Penny: <name>Penny</name>).
    2. Retirez en entier la section 'NPC' du PNJ contenant la ligne qui vous venez de trouver, commencant par la ligne contenant la dernière balise <NPC> rencontrée jusqu'à la ligne contenant la prochaine balise </NPC> .
  3. (optionnel) Retirer les données d'amitié d'un PNJ:
    1. Cherchez le nom de base du PNJ entre les différentes balises 'name' (par exemple pour supprimer Penny: <name>Penny</name>).
    2. Retirez en entier la section 'item' du PNJ contenant la ligne qui vous venez de trouver, commencant par la ligne contenant la dernière balise <item> encontrée jusqu'à la ligne contenant la prochaine balise </item>.
