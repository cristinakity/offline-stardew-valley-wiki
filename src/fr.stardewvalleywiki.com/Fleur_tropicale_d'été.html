Paquet

La Fleur tropicale d'été n'est utilisée dans aucun paquet.

Recettes

La Fleur tropicale d'été n'est utilisée dans aucune recette.

Couture

La Fleur tropicale d'été peut être utilisée dans la bobine du Machine à coudre pour créer une Jupe plissée.  

Quêtes

La Fleur tropicale d'été n'est utilisée dans aucune quête.

Notes

  • Si elle est placée dans la plage appropriée d'une Ruche (5 carrés dans n'importe quelle direction cardinale), une Fleur tropicale d'été rendra le miel produit plus rentable.
  • Les Fleurs tropicales d'été arrivées à maturité ont des chances égales d'être de l'une des cinq couleurs possibles.
 
  • Des Fleurs tropicales d'été de couleurs différentes occupent des emplacements différents dans l'inventaire ou dans les coffres. Six couleurs × trois niveaux de qualités = 18 espaces possiblement utilisés.
