Esta página explica cómo instalar SMAPI (la herramienta que le permite utilizar mods) en MacOS. Véase también:

  • Instrucciones para Android
  • Instrucciones para Linux
  • Instrucciones para Windows

Sumario

  • 1 Instalar
  • 2 Actualizar
  • 3 Desinstalar
  • 4 Solución de problemas

Instalar

  1. Instalar Mono. (La versión 'estable' está bien. Si esto no funciona, consulte la página de solución de problemas.)
  2. Descargue la última versión de SMAPI.
  3. Abra el archivo .zip que descargó para extraer su contenido.
  4. Expanda la carpeta 'SMAPI X.X.X installer' en tus descargas.
  5. Haga clic con el botón derecho en install on Mac.command y seleccione abrir.
  6. Seleccione abrir en el cuadro de diálogo y siga las instrucciones en pantalla.

¡Eso es todo! Simplemente inicie el juego para jugar con mods. No necesita cambiar los parámetros de lanzamiento de Steam (Eso solo es necesario en Windows).

Para instalar manualmente, consulte las instrucciones en el archivo readme del instalador.

Actualizar

Simplemente instale la nueva versión; el instalador limpiará automáticamente las versiones anteriores.

Desinstalar

  1. Inicie el instalador nuevamente (consulte los pasos de instalación).
  2. Cuando se le solicite, elija la opción uninstall.

Solución de problemas

  • Si el juego se congela al cargar un guardado sin error en la ventana SMAPI, intente reinstalar Mono (consulte el paso de instalación 1).
  • Si la ventana SMAPI ya no aparece después de una actualización del juego, reinstale SMAPI para arreglarlo.
  • Consulte la página de solución de problemas para la solución de problemas generales.
