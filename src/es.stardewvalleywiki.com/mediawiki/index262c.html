Formato

La llave de cada misión depende de la versión usada:

  • En 1.5.5 y anterior se usaba un integer positivo. Por esto, se recomendaba usar tu ID de nexus como prefijo (para evitar conflictos).
  • Desde la versión 1.6, se usa una cadena para la ID. Es recomendable usar el ID de tu mod como prefijo.

    • La entrada de cada misión contiene los siguientes campos (delimitados con una barra):

    Indice Descripción
    0 Tipo (e.g., Location, Basic, LostItem, etc)
    1 Titulo
    2 Detalles
    3 Pista/condición
    4 Solución
    5 Siguiente misión (-1 si no hay)
    6 Oro (0 si no se dará)
    7 Descripción de recompensa (Sólo si Oro no es -1. Aparentemente, no usado.)
    8 Cancelable
    9 (Opcional) texto de reacción

    Al usar el tipo de misión ItemDelivery, puedes especificar una cantidad luego de la ID de item.

    Ej: "5200": "ItemDelivery/Caramelo mágico/Abigail quiere que le traigas varias ametistas./Tráele 10 ametistas a Abigail./Abigail 66 10/-1/300/-1/true/Gracias, ¡se ve delicioso!"

    Aquí está la cadena en un formato fácil de editar:

    ID: Tipo/Título/Descripción/Pista/Solución/Siguiente misión (-1 si ninguna)/Oro (0 si nada)/Descripción de recompensa/Cancelable/Texto de reacción(opcional)

    Tipos de misión

    Existen múltiples tipos de misiones, cada uno con características diferentes. Los parámetros entre <> son obligatorios, pero los que usan [] son opcionales.

    Nombre Qué se usa en Solución Descripción
    Basic Nada Misiones base del juego, tienen código propio para detectar cuándo se cumplen.
    Building <ID de edificio> Se debe construir un edificio específico.
    Crafting <ID de objeto a construir> Se debe crear un objeto específico.
    ItemDelivery <PNJ> <ID de objeto> Se debe entregar un objeto a alguien.
    ItemHarvest <ID item> [cantidad] Se debe obtener un objeto.
    Location <ID de mapa> Se debe entrar a un lugar determinado.
    Monster <ID de monstruo> [cantidad] Se debe matar a un monstruo tantas veces como sea determinado.
    SecretLostItem <PNJ> <ID item> <amistad> <misión alternativa> Estas misiones no se muestran en el menú, son misiones secretas. Las puedes cumplir entregando el item secreto a alguien, y tienen un "final alternativo". En el juego, sólo se usa para el collar perdido de Caroline.
    Social Nada Especial, se usa en la introducción del juego.
