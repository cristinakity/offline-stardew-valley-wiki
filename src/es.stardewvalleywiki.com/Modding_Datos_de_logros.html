Esta página explica cómo el juego guarda y procesa datos de logros. Esta es una guía avanzada para desarrolladores de mod.

Datos puros

Los logros se guardan en Content\Data\Achievements.xnb, los que pueden ser descomprimidos para editar. Como ejemplo, esta es la información (de la versión 1.5.5):

Formato

Indice Campo Ejemplo
0 nombre Mejorando
1 descripción Mejora tu casa.
2 mostrar logro en colecciones antes de obtenerlo true
3 requiere logro previo -1
4 gorro ganado (índice de hats.xnb) 13

Si índice 2 es "true" e índice 3 es "-1", la estrella se mostrará en la colección (oscurecida si aún no se cumple). De lo contrario, índice 3 muestra el número del logro que debe ser ganado antes de que éste se muestre en Colecciones. (Ejemplo: debes obtener el logro 28 "Tesoro escondido" antes de que el logro 5 "Una colección completa" se muestre.)

El único logro con un valor de "false" en índice 2 es el logro secreto, que no se muestra hasta ser obtenido.

Para la lista de gorros, ve datos de gorro.
