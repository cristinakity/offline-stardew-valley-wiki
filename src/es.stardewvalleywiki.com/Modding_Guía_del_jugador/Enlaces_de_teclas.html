Algunos mods te permiten editar las combinaciones de teclas, que son botones del control/teclado/mouse que hacen algo en el juego. Esta página explica cómo configurarlos. (Esto funciona para cualquier mod que use las API estándar de SMAPI).

¿Cómo...?

Configurar los enlaces de teclas de un mod

  1. Instale el mod (consulte la ayuda).
  2. Ejecute SMAPI al menos una vez con el mod para permitirle generar su archivo de config.
  3. Edite el archivo config.json en la carpeta del mod con un editor de texto.
  4. Cambie las opciones de las teclas utilizando los valores a continuación. (En Mac, asegúrese de que no cambie a comillas rizadas)

Tenga en cuenta que algunos mods pueden no ser configurables.

Deshabilitar el enlace de teclas de un mod

Puede usar None para deshabilitar un enlace de teclas.

Encontrar el código para una tecla

Enlaces disponibles a continuación muestra todos los códigos de teclas disponibles, pero a veces no está claro cuál aplica (por ejemplo, para teclados que no están en inglés). Aquí se explica cómo verificar:
  1. Iniciar el Juego.
  2. En la ventana de la consola SMAPI, ingrese el comando test_input.
  3. En el juego, presiona las teclas que quieres probar.
  4. La consola SMAPI enumerará los códigos de las teclas que recibió.

Usar las combinaciones de teclas en Android

Puedes enlazar botones móviles físicos, pero no hay muchos disponibles. SMAPI en Android incluye un mod de teclado virtual que le permite agregar botones en pantalla que puede tocar para enviar entradas de teclado a sus mods instalados.

Para usar el teclado virtual:

  1. En el juego, debería aparecer un icono de teclado en la esquina superior izquierda de la pantalla.
  2. Toque el icono del teclado para mostrar los botones configurados.
  3. Toque el botón para enviar.

Para agregar una nueva tecla:

  1. Ejecuta el juego una vez para generar el archivo predeterminado config.json file.
  2. Abra el archivo Mods/VirtualKeyboard/config.json en un editor de texto.
  3. Agregue nuevos botones al campo buttons en el mismo formato.
    Consejo: cuando agregue un botón, establezca su valor X en el botón anterior + 96.

Teclas reservadas

No puede usar la tecla vinculada al cuadro de chat en el modo multijugador (predeterminadoT).
