武器

  • 语言
  • 监视
  • 编辑

武器可用于与怪物战斗。武器和其他工具不同，既不消耗能量，也不能在铁匠铺中升级。但是，大部分武器都可以通过消灭怪物获得，或从探险家公会购买，或在矿井中的宝箱中找到。大多数武器（除了弹弓）也可以卖给探险家公会。

大多数武器都有一个辅助攻击，通过右键点击触发。剑可以格挡一些伤害或击退飞行的敌人。棍棒可以猛烈敲击地面，击退附近的怪物并造成群体伤害。匕首可以快速进行连续戳刺。一些武器会有额外的加成。

尽管斧头、十字镐和镰刀并不是为战斗而设计的，但它们也可以用作武器并造成少量伤害。斧头和镐消耗能量，而镰刀不消耗。

剑

棍棒

棍棒是重型武器，比剑和匕首慢，但攻击范围很广。第二能力是猛击地面，并击退一大片区域的敌人。这种类型的攻击无法造成暴击。[1]

棍棒右击，并立即按住“C”键（默认设置），可以快速连续攻击。

棍棒的默认攻击速度为-8，即默认自带-4速度词条

要让弹弓装备上弹药，选取要使用的弹药，然后右键点击弹弓。

弹药类型决定了弹弓所造成的伤害，具体数值由以下公式计算[2]：

弹弓：
伤害 = 弹药系数 + [-(弹药系数 / 2), 弹药系数 + 1]之间的随机数
高级弹弓：
伤害 = 2 * (弹药系数 + [-(弹药系数 / 2), 弹药系数 + 1]之间的随机数)

使用最差的弹药（即：水果、蔬菜和蛋类）时，普通弹弓能造成1-3点伤害，高级弹弓能造成2-6点伤害。使用最好的弹药铱矿石时，普通弹弓能造成25-101点伤害，高级弹弓能造成50-202点伤害。

武器等级和卖出价格

武器等级以武器点数为基础（向下取整）:武器点数/7 + 1。 武器点数是用武器的几个统计数据来计算的，如平均伤害、速度、类型、附加防御和精准度、暴击率和暴击力量。

武器点数（四舍五入）= 平均武器伤害（四舍五入）x (1 + 0.03 x 速度{最小为 0;如果是匕首为 15 } + (精准度 / 2 + 防御) + ((暴击率 -0.02) x 200) + ((暴击力量 - 3) x 6) + 20{如果是黑暗之剑取该值，否则取0} + (防御 x 2) [3]

下面是一个应用该公式计算铱针等级的例子：(27 x (1 + 0.03 x 15) + 0 / 2 + 0 + ((0.10 - 0.02) x 200) + (7 - 3) x 6) + 0 + 0) / 7 + 1 (整数) = 12

售价是计算出的物品等级 × 100 / 2.[4]

小故障

  • 挥动剑、匕首或镰刀后，玩家角色会向前移动几个像素点。因此，站在地图出口处，向出口方向反复挥剑，即可进入任何地图区域周围的空间（例如，农舍周围的黑色区域）。
    • 要返回正常的地图区域，只需走回到出口点即可。
  • 在攻击动画结束之前，通过右键点击然后按左键（或C键），可以连续多次使用棍棒武器的二次攻击。这在用低级武器攻击高级敌人时很有用。
    • 过早地按下左键会导致正常的挥舞而不是二次攻击。
    • 如果在按左键/C时出现额外的动画，这个故障偶尔会使游戏软锁定。
  • 如果你在用剑格挡的同时与一个物体互动，比如一个灌木丛或一个村民，你会使你的精灵图（sprite）断裂，同时显示格挡的精灵图和面向屏幕的精灵图。你会像这样被卡住，直到你再次使用格挡。

趣闻

  • 有一个未实施的游戏代码，允许玩家通过邮件收到一把 "Battered Sword"，作为一个未知事件的奖励。 这把剑可以（通过未知的方式）升级4次，将其变为 "Hero's Sword"、"Holy Sword"、"Dark Sword"，最后变为 "Galaxy Sword（银河之剑）"。
  • 根据ConcernedApe的说法，圣剑可能从未在成品游戏中实现(1.6已失效)。
  • 用弹弓击中NPC会使与他们之间的友谊减少30点，并导致他们说“哎呀！我不敢相信你会那样对我！”或者“好痛！你干什么？”

历史

  • 1.4：弹弓现在可以放在电脑和主机版本的物品栏垃圾箱中（手机已经有这个功能）。在可获得的物品中增加了黑暗之剑。
  • 1.5：弹弓现在可以在解锁后在冒险者公会购买。在可获得的物品中添加了精灵之刃、股骨、长柄锤、暗影匕首、雪怪之牙。增加了额外的武器。改变了武器等级的计算。这导致大多数武器的等级和售价降低，棍棒类除外。修正了武器防御没有效果的错误。
  • 1.6：沙漠节期间可结婚对象会出售其专属武器。圣剑现在可获得了。
