创建角色

 
创建角色的界面

在一开始创建新游戏时以下选项是需要选择的：

  • 外貌 - 包括 24 种肤色、74 种发型、112 种上衣、4 种裤子和 31 种配饰。任何性别都可以使用，支持随机外貌的选择。这个并不影响游戏，以后可以更改，并且还有帽子和鞋类。
  • 名字/农场名字 - 这些仅限大部分的ASCII 可打印字符，包括空格、大小写字母、数字和`~!@#$%^&*()-=+[{]}\|;:'",<.>/?_字符。（但"不能使用。1.2 测试版起可以使用中文。）名字和农场名字有时会使用在对话中。玩家的名字以后可以更改，但农场的名字不能更改。
    • 两个字段都接受复制的文本（使用Ctrl + v粘贴文本）。
    • 名字的长度取决于其像素长度，并且允许的最大限度为文本框的长度。
    • 输入名字时不会改变字母的大小写（例如，若以全部小写的形式输入名字，它并不会自动首字母大写）。
  • 动物偏好 - 这将决定第一年春季获得的宠物的类型。有十个选项：五种狗，五种猫。
  • 最喜欢的东西 - 这将影响吃下星之果实后的文字显示。字符限制的文本框和名字/农场名字的是一样的。
  • 性别 - 这个对游戏的影响并不大，以后可以更改：
    • 在对话中人物角色将使用合适的代词。
    • 只能进入对应性别的温泉更衣室。（注：亚历克斯有时会在男更衣室里锻炼）
    • 结婚的 NPC 是没有性别限制的。与异性结婚后，女方将怀孕 14 天（并不影响速度或活动）并生下一个孩子。而与同性结婚则可以收养孩子。
    • 女性角色会收到爸爸的信，而男性角色会收到妈妈的信。

特殊字符

创建玩家角色时，在“名字”、“农场名字”或是“最喜欢的东西”的输入框内输入一些特殊字符会产生一些特别的声音或符号。

  • 输入以下字符会发出特别的声音。
    • 输入 < 会发出“叮”声
    • 输入 $ 会发出“叮铃”声
    • 输入 * 会发出敲击石头的声音
    • 输入 = 会发出“噗”声
    • 输入 + 会发出史莱姆移动的声音
  • 输入以下字符会显示特别的符号（游戏设置为中文时无效）
  • 输入 < 会变成 ♡ 符号
  • 输入 > 会变成 ▷ 符号
  • 输入 @ 会变成 ◁ 符号
  • 输入 $ 会变成 ⌾ 符号
  • 输入 * 会变成 💢 符号
  • 输入 = 会变成 ☆ 符号
 
  • 输入以下手柄按键时也会产生一些特别的声音或符号。
 

注释

  • 输入某些文本作为最喜欢的东西会触发一个彩蛋。
  • 一些特殊的按键如星号键将会触发某些声音。
  • 玩家在星露谷物语中没有固定性格特征。当在游戏中的事件和对话框里做选择时，玩家可以选择任何选项不会影响剧情，尽管有些选项会影响好感度。
  • 玩家可以和大多数的 NPC 成为朋友，没有互斥的友谊。（有些 NPC 如冈瑟是没有友谊值的。）
  • 虽然 NPC 有呼吸动画，但玩家没有。相反，玩家会眨眼睛。

漏洞

  • 仅 PC/移动版：可以使用方括号中的内部物品 ID 编号（英文）为角色起名，只要 NPC 提到该角色的名字，就会送给玩家与该编号相对应的物品。例如，输入 [645][499][163] 将分别获得一个铱制洒水器、上古种子和一条传说之鱼。（这个漏洞也适用于给孩子和动物起名。）

历史

  • 1.3.27：除了 PC 和移动版之外，其他所有版本都移除了物品 ID 命名漏洞。
  • 1.4：加入新发型和不同的裤子。
  • 1.5：加入更改名字和性别的功能，加入新发型。移除了使用物品 ID 命名漏洞获取错误物品。
  • 1.6：加入11种新配饰。加入四个新的动物偏好选项（两种狗和两种猫）。
