装身具

  • 言語
  • ウォッチ
  • 編集

装身具は装備可能な衣服アイテムです。それぞれがプレイヤーにユニークな恩恵を与えます。これらは戦闘の達人を取得した後に見つけることができます。ほとんどの装身具はモンスターがドロップしたり、鉱山・ドクロの洞窟・石切場の坑道にある木箱や樽で見つけたり、ドクロの洞窟のトレジャールームで入手したりできます。魔法のヘアジェルは例外で、砂漠フェスティバルのアレックスの店からのみ購入できます。

プレイヤーは同じ装身具を複数回入手できます。インベントリでスタックしません。冒険者ギルドにてdata-sort-value="1000">1,000gで売却できます。

バシリスクの足と魔法のヘアジェルを除くすべての装身具は、カナトコで再鍛造できます。再鍛造をすると、イリジウムののべ棒を1回につき3個消費して、ステータスがランダムに再決定されます。

ドロップチャンス

魔法のヘアジェルを除くすべての装身具は、約0.63%の確率でドクロの洞窟のトレジャールームで見つかります。[1]モンスターと木箱樽からのドロップ率の計算はより複雑で、いくつかの要因に依存します。[2]

モンスター

すべてのモンスターは倒されると、魔法のヘアジェル以外のランダムな装身具をドロップする可能性があります。その確率は次の式で計算されます：

{ (0.4% + 0.001% × モンスターの最大体力 + y) または 2.5% } の小さい方どちらかの値 + 毎日の運 / 25.0 + 幸運バフ × 0.133%[3]

式の中のyの値は、倒したモンスターによって決まります。

  1. 「グライダー（飛行する敵）」として分類される最大HPが150以上のモンスターの場合、y = 0.2%となります。この条件に該当するのは、イリジウムコウモリ、コウモリ（危険）、溶岩コウモリ、おばけガイコツ、おばけガイコツ（危険）、羽虫（危険）、炭素ゴースト、腐ったゴースト、スキッドキッド（危険）、昆虫（危険）、装甲虫（危険）、サーペント、ロイヤルサーペント、マグマおばけ、マグマスパーカー、ドワーフセントリー、青いイカです。[4]
  2. 「リーパー」に分類された場合、y = -0.5%となります。この条件に該当するのはクモのみです。
  3. それ以外の全てのモンスターではy = 0です。

(0.4% + 0.001% × モンスターの最大体力 + y)の式によって算出される確率は0.1%（クモ）から1.65%（18の体節を持つロイヤルサーペント）の範囲になるため、2.5%という上限値は特に影響を与えません。

装身具はモンスターのドロッププールに直接追加されるのではなく、別々に生成されます。これは、盗人の指輪やモンスター図鑑がドロップする装身具の確率や数に影響を与えないことを意味します。

木箱と樽

鉱山、ドクロの洞窟、石切場の坑道の破壊可能な木箱や樽は、魔法のヘアジェル以外の装身具をランダムでドロップする可能性があります。確率の計算式は以下の通りです。

｛ 0.4% + (その日の運 / 25.0) + (幸運のバフ × 0.133) } × { 1.0 + (effectiveMineLevel × 0.001) } × 0.8

effectiveMineLevelの値は次のように決まります：

  1. 鉱山では、effectiveMineLevel = 現在の階数
  2. ドクロの洞窟では、effectiveMineLevel = 現在の階数 + 120
  3. 石切場の坑道では、effectiveMineLevel = 5000

鉱山やドクロの洞窟の難易度は、木箱や樽からのドロップ率に影響しません。
