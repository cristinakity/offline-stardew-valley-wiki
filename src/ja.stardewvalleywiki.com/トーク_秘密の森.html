Lost Item Shop

"Lost Item Shop"に関して、公式の日本語名はあるのでしょうか？　なければWikiでの呼称として、適当な訳語をあてる必要があると考えています。私は「なくしものショップ」などゲームの雰囲気にも合う名称にならないか、と思っています。あるいは、もっと直訳でいくのであれば、「紛失アイテムショップ」などになるでしょうか。

(English)

Is there an official Japanese name for "Lost Item Shop"? If not, we would have to give it a Japanese name for this wiki. I think "なくしものショップ" might be a good name, which suits the game atmosphia ("なくしもの" is a plain expression of lost item). Or, we go with more literal translation, "紛失アイテムショップ" would be a choice ("紛失" means "lost", "アイテム" means "item". Being literal and using kanji, it might be a little stiff).
