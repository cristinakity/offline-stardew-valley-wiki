釣りは、釣竿で魚を捕まえたり、カニかごでアイテムを集めたりすることに関するスキルです。

概要と操作

 
釣りミニゲーム画面の普通の魚と伝説級の魚

釣竿をキャストするには、どうぐ使用ボタン（PCでは 、Xboxでは 、プレイステーションでは 、スイッチでは ）を長押しします。すると、どれだけ遠くに届くかを表すメーターが現れます。ボタンを離すとキャストします。キャスト中に移動キー（またはコントローラーのスティック）で着水位置を少しだけ動かすことができ、キャスティングの狙いをよりコントロールできます。これを使うと1マス"斜め"にキャストすることができ、普通では狙えない位置の泡を狙うことができます。

魚が食い付くと、プレイヤーの頭上に『！』マークが表示され、はっきりとした音が鳴ります。どうぐ使用ボタンを押してミニゲームを開始します。

ミニゲーム画面には、釣り糸の効果範囲を表す、動かすことのできる緑色の長方形があります。釣りミニゲームの目標は、右の釣りゲージがいっぱいになるまで、魚アイコンを緑色の釣りバーの内側にキープし続けることです。魚アイコンが釣りバーの上や下にはみ出ると釣りゲージが減少し、ゲージがゼロになると魚が逃げてしまいます。

どうぐ使用ボタンを押すことで緑色の長方形を上昇させることができます。連打すると、一番下でない限りは釣りバーを同じ位置に留めることができ、押しっぱなしにすると加速しながら上昇します。ただし、動かす際は注意が必要です。すぐに勢いがついて反対方向に戻すのに時間がかかるため、簡単に魚を通り過ぎて釣りゲージを減少させてしまいます。

釣り上げると、釣れた魚の名前とサイズが表示されます。コレクションメニューで、釣った魚の数と最大サイズを確認できます。

その種類の魚を初めて釣りあげた場合は「初ゲット！」と表示されます。

具体的な仕組み

魚の種類

見つけられる魚の種類は以下によって決定されます：

  • 季節
  • 天気（すなわち、晴れまたは雨）
  • 水域
  • 時刻

たとえば、森の湖、森の川、町の川にはそれぞれ独自の種類の魚が住んでいますが、いくつかは重複しています。完全なリストは釣りの戦略ページで見ることができます。

また、伝説級の魚は特定のマスにキャスティングしないと釣れませんが、一般的には、どのマスにウキを投げるかは重要ではありません。

バーの大きさ

緑色のバーは、釣りスキルが上がるにつれて大きくなります。ミニゲームの領域全体は568ピクセルあります。釣りレベル0では、バーの長さは96ピクセルです。釣りレベルが1上がると8ピクセル増加するので、レベル10では176ピクセルになります。  コルクのウキや  デラックス釣りエサを使ったり、達人のエンチャントや食べ物のバフで釣りレベルを10より高くすると、さらにバーを長くできます。最大では308ピクセルになり、これには釣りレベル10、  コルクのウキ2個、達人のエンチャント、  Qi調味料を使った  海の泡プリン、砂漠フェスティバルでシェフがエビを材料にして作った料理が必要です。砂漠フェスティバル以外では最大サイズは284ピクセルとなり、これはミニゲーム領域の総ピクセル数のちょうど半分です。

アップグレードされた釣竿（  ファイバーグラスの釣竿、  イリジウムの釣竿、  改良型イリジウムの釣竿）は、バーのサイズに影響しません。ただし、  練習用の釣竿を使用する場合、釣りレベルが5未満なら、バーは常に釣りレベル5と同等のサイズになります。

キャスティング距離

キャスティング距離の最大値は、プレイヤーの釣りスキルによって決まります。レベル0では、上下に最大3マス、左右に最大4マスまでキャストできます。レベル1、4、8、15になると、距離は1マス（全方向）ずつ伸びていきます。

スキルレベル15の到達は、料理、Qi調味料、および/または達人のエンチャントを付与した釣竿を組み合わせることで可能です。

実際のキャスティング距離は、ミニゲームのキャスティングメーターの満タン度に応じて、最大距離の何分の1かになります。最大距離の99%以上のキャスティングは「MAX」の文字で強調表示されます[1]。「MAX」のキャスティングには特別な特典はありません。

キャスティング距離より、浮きが着水する場所の方が重要です。長いキャスティングが釣果を向上させるのは、浮きが岸から遠く離れた場所に着水した場合のみです。水辺のギリギリに立って陸地から遠くにキャストするのはもちろん、他の近くの岸辺を避けることにも気をつけてください。たとえば川で釣りをするなら、できるだけ真っ直ぐで幅が広い場所を選ぶと良いでしょう。

魚が食いつく時間

魚が食いつく時間は、ウキが着水してから魚が食いつくまでにかかる時間です。基本値は0.6～30秒で、釣りレベルが1上がると、上限が0.25秒短くなります。1個装着するごとにスピナーは上限を5秒、ドレススピナーは10秒短くします。

ウキの着水後最初のアタリの場合（今回のキャスティングで他のアタリを見逃していない場合）は、下限/上限の両方が25%短くなります。

さらに、釣りエサ、マグネット、魔法のエサ、任意の専用のエサを使うと、下限/上限の両方が50%短くなります。ワイルドな釣りエサおよびチャレンジ釣りエサはこれとは別に25%短くなるので、エサなしの場合に比べて62.5%短くなります。デラックス釣りエサを使った場合は、エサなしの場合に比べて67%短くなります。

なお、下限は0.5秒より短くなりません。

  • 魚のサイズと品質（例：、ノーマル、シルバー、ゴールドなど）が良くなる可能性が高い。

パーフェクト

魚を緑のバーから出さずに釣り上げた場合、また、お宝キャッチャーを装着しておりかつ魚が緑のバーから出たのが宝箱回収中に限られる場合は、「パーフェクト！」と表示されます。パーフェクトキャッチには2つのメリットがあります。

  • シルバーまたはゴールド品質の魚の場合、魚の品質が1段階上がります。（例：シルバー品質はゴールド品質に、ゴールド品質はイリジウム品質になります。）
  • 魚を釣ったことによる獲得経験値が2.4倍になります。

魚のサイズ・品質

魚のサイズに影響するのは、釣りゾーンや釣りレベルで、そこにいくらかのランダム性が加わります。魚のサイズはfFishSizeという値で制御され、0.00では最小サイズ、1.00では最大サイズになります。[3]

すべての魚の最大/最小サイズはData\Fish.xnbにあります。最終的な魚のサイズは、minFishSize + (maxFishSize - minFishSize) * fFishSizeを切り捨てで求めます。パーフェクトキャッチの場合、これに1が足されます。パーフェクトキャッチではなく、計算結果がmaxFishSizeと同じだった場合、1が引かれます。これにより、ゲーム内に現れる最大サイズはData\Fish.xnbの値より1大きくなります。伝説級の魚については、パーフェクトキャッチではない場合の2つめの調整のため、最小サイズがData\Fish.xnbの値より1小さくなります。[3] これらの計算はすべてインチで行われ、画面に表示する際にセンチに変換されます。

基本の品質は、fFishSizeが0.33未満のとき普通、それ以外でfFishSizeが0.66未満のときシルバー、それ以外ならゴールドになります。この品質は上質なウキを使うと1段階上げることができます。また、最初の品質がシルバー以上なら、パーフェクトキャッチで1段階上がります。これらのブーストは重ねることができます。基本の品質は最高でもゴールドなので、イリジウム品質の魚を手に入れるには、パーフェクトキャッチをするか、上質なウキを使う必要があります。[3]

練習用の釣竿は、これらの計算を上書きして、釣れた魚をすべて普通品質にします。

泡のかたまり

泡

時々、泡のかたまりが水面に出現します。泡の場所へのキャスティングに成功すると、青いきらめきが現れます。泡には2つのメリットがあります[4]。

  • 魚の食いつきが4倍速くなります。
  • かかる魚の種類を判定する際に、有効な釣りゾーンが1増加します。つまり、ゴミが釣れにくくなり、釣れにくい魚が釣れる可能性が高くなります。伝説級の魚の中には、泡があると岸の近くで釣れるようになるものもあります。
    • この釣りゾーンの調整は、釣れる魚の種類にのみ適用されます。泡は魚のサイズや品質、宝箱にも影響を与えません。

泡はエリア（ペリカンタウン、シンダーサップの森など）ごとにに1つしか発生しませんが、複数のエリアで異なる泡が同時に発生する可能性があります。泡は釣竿で届かない場所に出現することもあります。リバーランド以外の牧場には、泡は出現しません[5]。

魚の活発化

魚の活発化は、日中（午後11時以前）に、ビーチ、シンダーサップの森、ペリカンタウン、山のいずれかでランダムに発生するイベントです。発生条件は以下のうち1つ以上が満たされていることです：

  • 少なくとも3匹の魚を釣り、4日間以上プレイしている。
  • 15日間以上プレイしている。

魚の活発化はお祭りの日には発生しません。お祭りのない日で発生条件が1つ以上満たされていれば、上記の4つのエリアのいずれかに泡が発生するごとに、1%の確率でその泡が魚の活発化になります。

魚が活発になっている間は、その場所にいる特定の魚（その日にその場所で釣れる魚）の泡が発生します。魚の活発化が発生すると、プレイヤーに対し「<場所>で<魚の名前>の行動が活発になっています。」というメッセージが表示されます。この泡の中で釣りをすると活発になっている魚だけが釣れます。伝説級の魚、伝説級の魚 II、その他の捕獲物は活発化の対象にはなりません。活発化の泡の効果は、通常の泡と専用のエサを組み合わせた効果と似ています。

  • 魚が食いつくまでの時間が半分になります。
  • 活発化している魚が、その場所で釣れる唯一の種類の魚になります。

魚の活発化は少なくとも90秒間続きます。90秒が経過すると、10%の確率で活発化は終わります。活発化が終わっていない場合は、ゲーム内時間10分ごとに活発化が終わるかどうかの判定が行われます。活発化が終わる確率は判定のたびに0.56%ずつ増加します。たとえば、午後2:00に始まった活発化がずっと続いていると仮定すると、午後5:10のタイミングで終わる確率は15.6%です。

エナジー

釣りのスキルレベルが0の時、釣竿をキャストすると8エナジーを消費します。それ以降は釣りのレベルが上がるごとに、使用するエナジー量は0.1ずつ減少していきます（表示は整数ですが、内部的には小数点以下も計算されます）。したがって、釣りレベル1で10回キャストすると、79エナジーが消費されることになります。 釣りレベル10では、すべてのキャスティングはエナジーを7消費します（表示上も内部的にも）。

キャスティングの調節

移動キーを使ってキャスティングの途中でラインの位置を少し変えることができ、キャストの狙いをより良くコントロールすることができます。キャスティングの調節は2つの場面で有効です。

  • 標準的ではない釣り場では、陸から離すためにキャストの角度を変える必要があります。
  • 泡が厄介な場所に出現している場合、浮きをその中に着水させるためにキャストの角度を変える必要があることがあります。

時間

  • シングルプレイの場合：
    • ミニゲーム中は時間が一時停止します。
      • ミニゲーム中にプレイヤーが攻撃を受けた場合（例：秘密の森で釣りをしている時にスライムに攻撃された場合）、時間が止まっていてもモンスターは攻撃を続け、プレイヤーに対してダメージを与え続けます。
      • バフの残り時間はミニゲーム中も減っていきます。
    • キャスティングメーターの表示中、ウキが飛んでいくアニメーション中、釣った魚が飛んでくるアニメーション中、釣った魚の種類がポップアップで表示されている間は、時間は一時停止しません。
    • 宝箱を釣り上げてそのインベントリを見ている間、時間は一時停止します。
  • マルチプレイヤーでは、時間は一時停止されません。
  • スターデューバレーまつりの釣り屋台、および氷まつりでの釣り大会中は、ミニゲーム中でもコンテストタイマーが一時停止することはありません。

経験値

ゴミや藻、海藻、ゼリー類を釣竿で釣るとそれぞれ3XPを獲得し、カニかごはの中身を回収するとそれが何であれ5XPを獲得します。魚のいる池から得られる経験値は魚のいる池を参照してください。

エサとウキまたは星の本を1冊読むたびに、250XPを獲得します。

それ以外の、魚を1匹捕まえるごとに獲得できる経験値の量は以下の通りです[6]：

XP = ((魚の品質 + 1) * 3) + (魚の難易度 / 3)

魚の品質は普通なら0、シルバーなら1、ゴールドなら2、イリジウムなら4です。
魚の難易度は5から110までの数字で、データファイルFish.xnbに記載されています。

例えば、普通の品質のイワシ（難易度30）であれば、((0 + 1) * 3) + (30 / 3) = 13XPとなります。ゴミもこの式に従い、品質と難易度は0で、基本の3XPだけが残ります。

XPは、宝箱を釣り上げた場合は2.2倍、「パーフェクト」の場合は2.4倍、伝説級の魚の場合は5倍になります。これらの掛け算は積み重ねられ、順番に実行されます（最初に宝箱、次にパーフェクト、そして伝説級の魚）。計算の途中で整数でなくなれば、次の計算の前に切り捨てられます。

たとえば、宝箱と一緒にパーフェクトキャッチでイワシを捕まえると、13 * 2.2 = 28.6 （28に切り捨て）、28 * 2.4 = 67.2（67に切り捨て）となり、67XPを獲得できます。

パーフェクトキャッチをするとシルバー品質の魚はゴールド品質に、ゴールド品質の魚はイリジウム品質になりますが、XPの計算に使われるのは元々の品質です。

たとえば、パーフェクトキャッチでイリジウム品質のイワシを捕まえると、元々のゴールド品質として計算され、((2 + 1) * 3) + (30 / 3) = 19、19 * 2.4 = 45.6（45に切り捨て）となり、45XPを獲得できます。

別の例として、宝箱と一緒にパーフェクトキャッチでゴールド品質のクリムゾンフィッシュ(難易度95)を捕まえると、((1 + 1) * 3) + (95 / 3) = 37.67（37に切り捨て）、37 * 2.2 = 81.4（81に切り捨て）、81 * 2.4 = 194.4（194に切り捨て）、194 * 5 = 970となり、970XPを獲得できます。

カニかごから回収したり釣竿でアイテムを釣ったりして経験値を得ると、必要経験値に達すればすぐにレベルが上がりますが、「レベルアップ」のウィンドウは寝るまで出てきません。

宝箱

釣りミニゲームの最中、開始後1～3秒の間に釣りバーを動かすエリアに宝箱が現れることがあります。宝箱が見つかる基本確率は15%です。マグネット (+15%) やお宝キャッチャー (+5%)、海賊の特化職 (+15%) によって合計50%まで確率を上げることができます。さらに、その日の運によって+5%～-5%上下し、特別な飾りがあれば1.25%上がります。食べもの/飲みものの幸運バフは、1ポイントあたり+0.5%です。[7]スターデューバレーまつりや氷まつりでの釣りでは宝箱は出現しません。

宝箱には独立した進捗ゲージがあり、宝箱アイコン を緑色の釣りバーの中にキープすると進めることができます。宝箱を取ると、魚が釣れた後にランダムでボーナスアイテムを獲得できます。魚が逃げると宝箱も逃してしまいます。

魚と宝箱を捕まえることに成功すると、宝箱のアイテムがメニューに表示されます。持ち物が一杯の状態で魚と宝箱を釣ると、釣った魚は宝箱の中身と一緒に表示されます。すべてのアイテムを持ち物に入れる前にメニューを閉じると、残りのアイテムは失われます。

宝箱の中には複数のアイテムが入っていることがあります。複数のアイテムが入っていた場合、同じアイテムが複数のスロットに入っていることもあります。表に記載された個数は1スタックあたりの個数です（たとえば、紛失図書は1スタックに1つですが、2スタックに分かれて2つ現れることもあります）。

幸運は、宝箱の中身に少し影響します。毎日の運は、以下の表の確率で ≈ がついた特別なアイテムの確率を高めます。幸運バフは、1スタックに含まれる個数を増やします（特に、資源やジオードの個数が倍になる確率を高めます）。また、幸運バフは、良い指輪が現れることを可能にします（磁石の指輪Sの代わりに磁石の指輪、光の指輪Sの代わりに光の指輪）。

金の宝箱

釣りの達人を取得すると、釣りをしたときに宝箱が 金色になる場合があります。宝箱が金色になる基本確率は25%です。これは毎日の幸運の値によって調整され、最高で10%増加、最悪で10%減少し、特別な飾りで2.5%が追加されます[7]。そのため、その日の運勢が最高かつ特別な飾りを持っている状態では、金の宝箱が釣れる確率は最大の37.5%になります。

金の宝箱には通常の宝箱よりも平均して多くのアイテムが含まれています。魚卵や『海の宝石』といった複数のスロットに出現しないアイテムを除き、アイテムの数が増える確率は27.4%です。加えて、Qiビーンや金の動物ビスケット、魚卵のような少数の例外を別にすれば、それぞれのアイテムは50%の確率で以下のアイテムに変化します[8]。

実績

釣りに関連する実績は4つあります。緑の藻、白い藻、海草、海ゼリー、川ゼリー、洞窟ゼリーも実績にカウントされます。

  • 釣り人（魚を10種類釣りあげる）
  • 海の古兵（魚を24種類釣りあげる）
  • 釣りマスター（魚を全種類釣りあげる）
  • 大漁旗（魚を100匹釣りあげる）

バグ

  • iOS版には、ほとんどの場所の釣りゾーンが変更されるバグがあります。釣りゾーンが、実際の浮きの位置より左上に1マスずれた座標で計算されます。これにより、下や右へのキャスティングは本来のゾーンより低い値になり、上や左へのキャスティングは本来のゾーンより高い値になります。このバグは魚のサイズや品質、宝箱の中身に影響しますが、伝説級の魚が釣れる場所には影響しません。
