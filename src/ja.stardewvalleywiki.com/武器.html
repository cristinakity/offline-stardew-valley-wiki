武器

  • 言語
  • ウォッチ
  • 編集

武器はモンスターと戦うのに用います。武器は他のどうぐと違って使用時にエナジーを消費せず、鍛冶屋でアップグレードすることもできません。ほとんどの武器はモンスターからのドロップ、冒険者ギルドでの購入、鉱山の宝箱から入手できます。パチンコ以外の武器は冒険者ギルドに売却することができます。

ほとんどの武器では、右クリックでスペシャルアタックができます。剣では、いくらかのダメージをブロックするか飛んでくる敵をノックバックします。クラブは地面を叩くことで近くの敵をノックバックし、加えて軽微なダメージを与えます。短剣は素早い連撃を繰り出すことができます。

戦闘用に作られたものではありませんが、オノ、ツルハシ、カマはわずかなダメージを与える武器として使えます。オノとツルハシはエナジーを消費するのに対し、カマは消費しません。

クラブ

クラブは重い武器で剣や短剣より遅いですが、広範囲に攻撃ができます。スペシャルアタックでは目の前の地面を叩きつけ、強いノックバックを与える範囲攻撃をします。この攻撃はクリティカルになりません。[1]

右クリックをして即座に左クリックを連打することで、短時間で複数回の攻撃ができます。

パチンコ

パチンコによるダメージは主に使用した弾によって決まります。マスターパチンコは同じ弾を用いた普通のパチンコに比べて、一般的に2倍のダメージを与えることができます。

パチンコの弾

弾をパチンコに入れる際には、弾を選択し、パチンコの上で右クリックを押します。

与えられるダメージは、次の公式によって決まります。

パチンコ:
ダメージ = 弾薬乗数 + [-(弾薬乗数 / 2), 弾薬乗数 + 1]のランダムな数
マスターパチンコ:
ダメージ = 2 × (弾薬乗数 + [-(弾薬乗数 / 2), 弾薬乗数 + 1]のランダムな数)

くだもの、野菜、タマゴなどの最悪の弾を用いた場合、パチンコなら1-3のダメージ、マスターパチンコなら2-6のダメージを与えます。最高の弾であるイリジウム鉱石を用いた場合、パチンコなら25-101のダメージ、マスターパチンコなら50-202のダメージを与えます。

武器のレベルと売価

武器のレベルは、武器のポイント / 7 + 1の結果を小数点以下切り捨てで求めることができます。武器のポイントの計算方法は以下の通りです。
average weapon damage (rounded down) x (1 + 0.03 x (speed{min 0} + 15{if dagger, else 0})) + precision / 2 + defense + (crit. chance - 0.02) x 200 + (crit. multiplier - 3) x 6 + 20{if darksword, else 0} + defense x 2[2]

例としてイリジウムニードルのレベルは以下のように計算されます。
(27 x (1 + 0.03 x 15) + 0 / 2 + 0 + (0.10 - 0.02) x 200 + (7 - 3) x 6 + 0 + 0 x 2) / 7 + 1 → 12

売価はレベル x 100 / 2です。[3]

グリッチ

  • マップの出口ギリギリに立ち、繰り返し剣を振ることで、マップの周囲の空間（例えば、牧場の周辺の黒い領域)に抜けることが可能です。
    • 通常は入れない領域から戻るには、シンプルに出口の方に歩いてください。
  • 攻撃のアニメーションが終了する前に右クリックしてから左クリック（またはCキー）を連打することで、クラブ武器のスペシャルアタックを数回立て続けに使用することが可能です。
    • 左クリックを早く押しすぎるとスペシャルアタックではなく通常のスイングになります。
    • 左クリック/Cキーの連打中に新しいアニメーションが発生すると、ゲームがフリーズします。
  • 茂みや村人などのオブジェクトを操作しながら同時に剣でブロックすると、スプライトが破壊されます。ブロックを再度使用すると元に戻ります。

トリビア

  • 未知のイベントの報酬として手紙で"Battered Sword"が受け取れる未実装のゲームコードがあります。この剣は4段階に分けてアップグレードでき、"Hero's Sword"、"Holy Sword"、"Dark Sword"、"Galaxy Sword"と変化していきます。
  • ConcernedApeによると、聖なるつるぎが製品版のゲームに実装されたことは一度もないようです。

更新履歴

  • 1.4: PC版とコンソール版でパチンコをインベントリのゴミ箱に入れることができるようになりました（モバイル版にはすでにこの機能があります）。入手可能なアイテムに闇の剣を追加しました。
  • 1.5: パチンコは、ロックを解除した後、冒険者ギルドで購入できるようになりました。入手可能なアイテムにエルフの剣、大たい骨、カジェル、シャドウダガー、イエティのキバを追加しました。追加の武器を追加しました。武器のレベルの計算方法を変更し、結果的にクラブ以外のほとんどの武器のレベルと売値が下がりました。武器の防御が効果を発揮しないバグを修正しました。
