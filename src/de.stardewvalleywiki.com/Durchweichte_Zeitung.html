Durchweichte Zeitung
Das ist Müll.
Information
Quelle Angelrute • Krabbenreuse • Mülleimer • Fischteich
Jahreszeit  Alle
Erfahrungs­punkte
  • Angelrute: 3 Fischen EP
  • Krabbenreuse: 5 Fischen EP
  • Fischteich: 10 Fischen EP
Verkaufspreis data-sort-value="0">0 G

Die Durchweichte Zeitung gehört zum Müll. Sie kann an allen Angelplätzen das ganze Jahr über gefangen werden. Sie kann auch beim Durchsuchen von Mülleimern gefunden werden. Mit einer Recycle-Maschine kann man aus ihr nützliche Dinge gewinnen. Wird die Durchweichte Zeitung beim Fischen gefangen, bekommt man durch sie 3 Erfahrungspunkte. Wenn sie in einer Krabbenreuse gefangen wird, gibt sie 5 Erfahrungspunkte.

Bündel

Die durchweichte Zeitung kann zu Stoff recyclet werden, was wiederum im   Handwerker-Bündel in der Speisekammer verwendet wird.

Rezepte

Sie wird in keinem Rezepten verwendet.

Schneidern

Man kann die Durchweichte Zeitung in der Spule der Nähmaschine verwenden, um das Mülltonnenhemd aus ihr zu schneidern.  

Aufträge

Sie wird in keinem Auftrag gebraucht.
