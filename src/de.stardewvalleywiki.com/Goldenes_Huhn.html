Goldenes Huhn
Gut versorgte, ausgewachsene Hühner legen jeden Tag Eier. Lebt im Hühnerstall.
Information
Gebäude Großer Hühnerstall
Kaufpreis N/A
Erzeugnis Gold-Ei

Das Goldene Huhn ist ein Tier, welches erhalten werden kann, wenn ein Gold-Ei in einen Brutkasten gelegt wird. Das erste Gold-Ei kann auf 3 verschiedene Arten erhalten werden, nachdem 100% Perfektion erreicht wurde::

  • Marnies Ranch für data-sort-value="100000">100.000 G.
  • Geheimes Walnusszimmer von Mr. Qi für data-sort-value="100"> 100.
  • Zufälliges Ereignis welches stattfinden kann nachdem der Spieler schlafen geht. Eine goldene Hexe fliegt über den Hühnerstall und hinterlässt ein Gold-Ei.

Ein Gold-Ei benötigt 9000m (6.25d) zum ausbrüten, bzw. 4500 (3.125d) mit der Stallmeister Fähigkeit. Ein Goldenes Huhn erreicht 3 Tage nach dem Schlüpfen das Erwachsenenalter.

Produkte

Das Goldene Huhn kann jeden Tag ein Gold-Ei produzieren (wenn es am Vortag gefüttert wurde). Diese erzielen einen Verkaufspreis von mindestens data-sort-value="500"> 500 G. Das Gold-Ei kann beim Kochen anstelle anderer Eier verwendet werden. Gold-Eier können auch in eine Mayonnaise-Maschine verarbeitet werden, um 3 Gold-Qualität Mayonnaise zu produzieren, welche sich für je data-sort-value="285"> 285 G verkaufen lassen.

Verkauf

Goldene Hühner können genauso wie normale Hühner verkauft werden. Je mehr Herzen ein Tier besitzt, desto höher der Verkaufspreis. Ein neugeborenes Küken bringt data-sort-value="252"> 252 G, und der Verkaufspreis liegt bei Goldenen Hühnern mit maximalen Herzen bei data-sort-value="1040"> 1.040 G, genauso wie bei normalen Hühnern.
