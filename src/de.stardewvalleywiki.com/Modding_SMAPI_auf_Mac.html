Diese Seite erklärt die Installation von SMAPI (das Programm zum Laden von Mods in Stardew Valley) auf MacOS. Siehe auch:

  • Anleitung für Linux
  • Anleitung für Windows

Inhaltsverzeichnis

  • 1 Installation
  • 2 Aktuallisieren
  • 3 Deinstallieren
  • 4 Spezielle Anmerkungen für MacOS

Installation

  1. Installiere Mono. (Die "stable channel" Version sollte funktionieren. Wenn nicht siehe Problembehebung.)
  2. Lade die Version von SMAPI herunter.
  3. Entpacke die .zip Datei die du heruntergeladen hast.
  4. Aktiviere das Installationskript:
    1. Öffne das Launchpad.
    2. Tippe Terminal ein und wähle die "Terminal" App aus.
    3. Führe diesen Command aus: 
      mdfind -0 'kMDItemFSName = "install on Mac.command" && kMDItemContentType = "com.apple.terminal.shell-script"' | xargs -0 xattr -d com.apple.quarantine
  5. Doppelklicke auf install on Mac.command und folge der gegebenen Anleitung.

Das wars! Starte das Spiel nun um mit Mods zu spielen Du must keine Einstellungen auf Steam vornehmen (das ist nur auf Windows erforderlich).

Um die Installation manuell vorzunehmen, lies dir die beim Installer beinhaltete readme Datei durch.

Aktuallisieren

Installiere einfach die neue Version so wie oben beschrieben, der Installer entfernt ältere Versionen automatisch.

Deinstallieren

  1. Öffne die install on Mac.command Datei.
  2. Wähle "uninstall SMAPI" aus.

Spezielle Anmerkungen für MacOS

  • Du musst nach einem Update für MacOS eventuell Mono neu installieren.
  • Du musst eventuell SMAPI nach einem Update für das Spiel neu installieren
