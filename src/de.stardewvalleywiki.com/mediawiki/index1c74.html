Bietet eine mäßige Menge Licht.

Das Lagerfeuer ist eine herstellbare Beleuchtung. Es spendet Licht in der Nacht. Rechtsklick auf die Feuerschale macht das Feuer an oder aus.

Trivia

  • Das Lagerfeuer außerhalb von Linus' Zelt kann an oder aus geschaltet werden ohne Linus Dialog oder Verhalten zu beeinflussen.
