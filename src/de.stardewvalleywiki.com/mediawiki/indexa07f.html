Die Süße Edelsteinbeere ist eine Nutzpflanze die aus einer Seltenen Saat in 24 Tagen heranwachst. Sie kann beim Reisewagen für data-sort-value="3000"9.000–15.000 G gekauft werden.

Eine Seltene Saat kann im Frühling & Sommer beim Reisewagen für data-sort-value="1000">1.000 G gekauft werden, oder für data-sort-value="200"600–1.000 G in jeder Jahreszeit. Sie kann auch aus einer Süßen Edelsteinbeere im Saatguterzeuger produziert werden.

Trotz ihres hohen Preises, wird die Beere von allen Bewohnern im Sternentautal als ein neutrales Geschenk betrachtet. Wird sie hingegen dem Altmeister Cannoli in den Geheimen Wäldern gegeben, wird man als Belohnung eine Sternenfall-Frucht bekommen, und die Energie wird permanent ansteigen.
