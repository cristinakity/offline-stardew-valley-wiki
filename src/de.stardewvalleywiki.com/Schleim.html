Schleim
Ein schimmernder, gallertartiger Blob ohne Geruch.
Information
Quelle Schleime • Krobus • Schleimball • Reisewagen • Fischteich
Verkaufspreis data-sort-value="5">5 G
Diese Seite ist für das fallen gelassene Objekt. Für die Monster, die dieses Objekt fallen lassen, siehe Schleime.

Schleim ist eine Monsterbeute von Schleimen. Tötet man einen Schleim, erhält man 1-2 Schleim. Man kann sie auch von einem Schleimball erhalten oder Montags von Krobus für data-sort-value="10">10 G kaufen (maximal 50) . Es kann zufällig im Reisewagen für data-sort-value="15"100–1.000 G erscheinen. Ein Schleimmakrelen Fischteich kann ebenfalls täglich 10, 30 oder 50 produzieren, nachdem die Anzahl mindestens 9 erreicht hat.

Schneiderei

Schleim kann in der Spule der Nähmaschine verwendet werden, um ein grünes T-Shirt   herzustellen.

Aufträge

  • 20 Schleim werden von einer Schleimmakrele in einem Fischteichauftrag verlangt, um die Kapazität von 1 auf 3 zu erhöhen.
  • 5 Schleim können von Zwergwelsen, Forellenbarschen oder Mitternachtskarpfen in Fischteichaufträgen verlangt werden, um die Kapazität von 3 auf 5 zu erhöhen.

Geschichte

  • 1.4: Robuster Ring benötigt jetzt 25 Schleim für die Herstellung. Schleim kann jetzt in der Schneiderei verwendet werden. Kann in Fischteichaufträgen verlangt werden. Kann in Fischteichen produziert werden.
