Palme
Information
Samen Keine
Zapfhahnerzeugnis Keine
Wachstumsdauer unterschiedlich

Die Palme ist eine Baumart, die auf der Ingwerinsel und in der Calico Wüste gefunden werden kann. Es fallen keine Samen vom Baum, somit können Palmen nirgendwo eingepflanzt werden. Palmen können gefällt werden und es kann sein, dass dabei oder beim Schütteln der Palme eine Kokosnuss auftaucht. Es besteht eine 20%ige Chance, dass die Palme jeden Tag um eine Stufe wächst. Zapfhähne können nicht bei Palmen angebracht werden.
