Multijogador

  • Língua
  • Vigiar
  • Editar
Uma captura de tela do modo multijogador mostrando duas cabines e dois jogadores.

Multijogador é um modo de jogo opcional no qual 1 a 8 jogadores podem jogar no mesmo mundo pela rede de área local [1] ou pela internet. Cada jogador precisa de sua própria cópia do jogo (a tela dividida não é suportada), mas os jogadores do GOG e do Steam podem fazer cross-play com outra pessoa[2]. O jogador que criou o mundo tem o arquivo salvo, pode lançá-lo no modo multiplayer[3],e pode voltar para o modo de um jogador a qualquer momento. Outros jogadores podem se conectar via Steam (suporte não-Steam está planejado, mas ainda não anunciado).[4]

O jogador que criou a fazenda (o "host") tem o arquivo salvo em seu dispositivo, pode abrir o jogo no modo multijogador, e pode jogar sozinho a qualquer hora. Se o host não está online ou a fazenda não está atualmente aberta para outros jogadores, o mundo não estará acessível para outros jogadores que tinham se conectado ao mundo previamente e eles não podem ver seus personagens nem qualquer outra informação sobre aquele mundo. A posse da fazenda não pode ser transferida para outros jogadores sem o uso de modificações no jogo.

Restrições

  • No console apenas:
    • O multijogador online precisa de uma assinatura dos serviços online desse console. (Não é necessário para o modo tela dividida local)
    • Não é possível ter jogadores de tela dividida e multijogador online ao mesmo tempo.
    • No Nintendo Switch apenas, o modo tela dividida é limitado para apenas dois jogadores. Cada jogador precisa de um controle completo ou dois Joy Cons.
  • O modo multijogador entre plataformas (e.g., PS4 ↔ PC) não é suportado, exceto entre versões de computador (e.g., Windows ↔ MacOS).

Criando um mundo multijogador

Mundos multijogadores podem tanto serem criados com a intenção do multijogador ou serem convertidos de um arquivo salvo no modo de Um Jogador e aberto para novos jogadores entrarem no mundo.

Começando um novo mundo multijogador

Para começar um mundo feito para o multijogador:

  1. Na tela inicial, escolha Coop. ao invés de Novo.
  2. Clique na aba Host.
  3. Clique em Criar Nova Fazenda...

O host pode optar por adicionar quantas cabanas ele deseja. O número de cabanas no início determina quantos cooperadores podem se juntar imediatamente ao jogo assim que a conexão estiver aberta. Cabanas podem ser adicionadas ou removidas assim que a fazendo for criada.

A margem de lucro não pode ser mudada depois da criação da fazenda, já o compartilhamento de dinheiro pode.

Começar no modo tela dividida

  1. Crie ou carregue o mundo normalmente.
  2. Construa uma cabana para cada novo jogador no modo Um jogador.
  3. Conecte um controle ao computador/console para cada jogador extra. (No computador, o teclado estará sempre reservado para o primeiro jogador, apesar de que o primeiro jogador possa usar um controle também.)
  4. Abra o menu dentro do jogo e escolha a aba Opções.
  5. Desça até a seção Multijogador e escolha Começar Co-op Local.
  6. Cada jogador extra precisa pressionar start no controle conectado.

Caso o host tenha carregado o mundo pelo Coop. na tela inicial, será possível jogar com jogadores remotos e jogadores de tela divida no mesmo mundo (Sujeito às restrições).

Convidando amigos para um mundo Um Jogador

  1. Construa uma cabana para cada novo jogador no seu mundo Um Jogador.
  2. Salve e saia para a tela inicial.
  3. Abra o mundo pela aba Coop. ao invés de Carregar para permitir que outros jogadores entrem.

Conectando-se

Assim que o mundo estiver pronto para o multijogador, o host deve abrir o mundo para outros jogadores poderem entrar. Todos os mundos são fechados para outros jogadores por padrão assim que ele for criado. Para abrir o mundo, o host deve ir para a aba de "Opções" em seu Inventário e selecionar as preferências que eles querem usar para abrir o mundo.

Para cooperadores entrarem:

PC
No Linux/MAC/Windows, você pode se conectar com outros jogadores de três diferentes maneiras:
  • Quando rodar pela Steam, fazendas multijogadores criadas por seus Amigos Steam, vão aparecer na tela de entrada do "Co-op".
  • Quando rodar pelo GOG Galaxy ou pela Steam, um código de convite aparece nas opções dentro do jogo. Outros jogadores que também tenham GOG Galaxy ou Steam podem digitar esse código na tela de entrada para entrar no mundo.
  • Via endereço de IP. Esse método pode precisar de um firewall aberto ou um serviço de LAN virtual, como por exemplo, Hamachi.
    • Por padrão, a porta dos servidores é UDP porta 24642.
PlayStation 4 (PS4)
No PS4, você pode se conectar com outros usuários através da PlayStation Plus. Multijogador sem uma assinatura não é suportado.
Nintendo Switch
No Nintendo Switch você pode se conectar com outras pessoas de duas maneiras diferentes:
  • Através dos amigos na sua lista de amigos. A assinatura Nintendo Switch Online é necessária.
  • Através de um jogo local. (Não é necessário a assinatura)
Xbox One
No Xbox One você pode se conectar com outros usuários através da XboxLive Gold. Multijogador sem uma assinatura não é suportado.

Características

Multiplayer é praticamente idêntico ao single-player. Aqui estão alguns recursos específicos para o modo multiplayer:

Fazenda Quatro Cantos

 
A fazenda quatro cantos

A Fazenda Quatro Cantos foi feita para o uso em um mundo multijogador. A fazenda é dividida em Quatro Cantos, onde em cada um deles, se manifesta alguma individualidade dos outros mapas especiais presentes no jogo, além de um pequeno lago de água em cada um deles, com exceção, do quadrado inferior esquerdo, onde temos um lago maior de água. Mesmo sendo feito para o multijogador, a fazenda está disponível para o modo de Um Jogador.

Cabanas

A fazenda pode ter até 3 cabanas (selecionadas durante a criação da fazenda). Para cada uma delas, um jogador pode se juntar ao mundo. Caso a fazenda comece com menos de 3 cabanas (inclusive se a fazenda foi criada originalmente como um mundo Um Jogador), a Robin pode construir até 3 cabanas na fazenda por data-sort-value="100"> 100 ouros cada. A construção de cabanas ocorre instantaneamente, ou seja, cabanas extras podem ser construídas logo no primeiro dia de jogo.

O host também pode demolir uma cabana a qualquer momento. Demolir uma cabana remove seu ocupante do jogo.

Uma cabana pode ser melhorada na carpintaria pelo cooperador que a ocupa, porém não pode ser melhorada pelo host ou por outros players. O layout interior de uma cabana melhorada é idêntico ao layout da casa principal melhorada, com apenas uma exceção: a fogueira é substituída por um armário que fica com os itens do cooperador enquanto o mesmo não estiver online. Esse armário não pode nem ser movido nem acessado pelo cooperador que o possui. Assim como a casa principal, o tamanho da cabine não muda quando é melhorada.

Margem de lucro

Ao criar um novo mundo multijogador, você pode escolher uma margem de lucro de 25%, 50%, 75% ou a margem padrão (100%). Optar por margens mais baixas ajuda a balancear a economia no modo multijogador, reduzindo o ouro obtido ao vender a maioria dos itens (incluindo lavouras, coletas, minerais e comidas cozidas) para balancear o aumento na produtividade devido ao crescimento no número de jogadores ativos. Por exemplo, com uma margem de lucro de 25%, você pode vender Trigo por data-sort-value="6"> 6 ouros ao invés de data-sort-value="25"> 25 ouros.

O preço de compra de sementes no Armazém do Pierre e o preço de compra para Mudas de Grama, Açúcar, Farinha de Trigo e Arroz no Mercado Joja são escalados conforme a margem de lucro escolhida pelo host. Os preços de outros itens (incluindo itens do Ferreiro e da Peixaria além de construções e melhorias de ferramentas) e as recompensas em ouro das missões não são afetados.

Diminuir a margem de lucro deixa o ouro muito mais difícil de se conseguir cedo no jogo; progredir nas suas habilidade igualmente, produzir itens e completar missões do Precisa-se de ajuda! se tornam muito mais importantes. Por exemplo, mesmo com a margem de lucro reduzida, Willy ainda vende seus covos por data-sort-value="1500"> 1 500 ouros, então minerar ferro e produzir seu próprio covo pode se tornar mais econômico.

Note que não há uma opção de margem de lucro ao criar um mundo no modo Um Jogador. Mesmo assim, você pode ter um jogo no Um Jogador com uma margem de lucro diferenciada. Basta criar um salvamento no modo multijogador e jogar sozinho. Escolha "Co-op" no menu principal, crie seu mundo, após isso, vá em "Carregar" e jogue esse mundo no modo de Um Jogador.

Casamento

Jogadores podem se casar com um aldeão normalmente, que funciona do mesmo modo que o casamento no modo Um Jogador. Cada aldeão só pode ser casado com um jogador, e os filhos serão adicionados na casa do jogador casado.

Jogadores podem se casar com outros jogadores dando à eles um anel de casamento. Jogadores casados recebem uma Fruta-estrela em uma caixa de presentes roxa ao lado das suas camas originais após o casamento (equivalente a Fruta-estrela recebida pelo Aldeão cônjuge). Uma casa é elegível para crianças se um dos jogadores casados possuírem a melhoria que dá acesso ao quarto de crianças, lembrando que os jogadores não podem já terem dois filhos e devem estar dormindo na mesma casa juntos; o jogo vai aleatoriamente deixar um dos jogadores propor ter um filho para o outro, e se for aceito, a criança vai ser adicionada à casa que os dois jogadores estão dormindo (se cumprir todos os requerimentos), caso contrário na casa do outro jogador (se cumprir todos os requerimentos), caso contrário, na própria casa.

Gameplay

  • Algumas decisões são tomadas de forma colaborativa por todos os jogadores, clicando em diálogos ou juntando áreas:[4]
    • quando dormir, entrando em uma cama;
    • quando começar um festival, entrando na área;
    • se deve terminar o Centro Comunitário ou comprar uma associação Joja;
    • quando pausar o jogo usando o comando /pause comando (abrir um menu, comer ou o minijogo da pesca não pausa no modo multijogador).[5]
  • Todos os jogadores compartilham o mesmo dinheiro e terras agrícolas, porém a partir da atualização 1.4 o host pode escolher se as finanças da fazenda serão individuais ou compartilhadas entre os fazendeiros.
  • Na residencia do prefeito podem ser feitas algumas ações, como transferir dinheiro para outro fazendeiro, ou para o host, recolher todas as economias dos fazendeiros.
  • Cada jogador tem sua própria barra de energia [6], suas próprias Habilidades, e suas próprias relações com NPCs [4].
  • Cada NPC só pode ser casado por um jogador. Se vários jogadores querem o mesmo cônjuge, o primeiro a propor em casamento ganhará a sua mão..[7]
  • Caso o Jogador deite na cama, ele começa a recuperar sua energia. Não há como fazer isso no modo de Um Jogador.
  • Interações entre o jogador:
    • Os jogadores podem se casar e ter filhos. Os jogadores propõem um ao outro usando um anel de casamento artesanal.[4][8] A receita do anel de casamento pode ser obtida no Carrinho de Viagem por data-sort-value="500"> 500 ouros, e então permitirá que o jogador crie um anel de casamento com 5 Barras de Irídio e um Fragmento Prismático.
    • Os jogadores podem dançar uns com os outros na Dança das Flores (sem requisitos anteriores).[9]
    • Os jogadores podem ver a pontuação dos outros jogadores em certos festivais.

Missões e progresso

  • A progressão em missões são mantidas em um teor individual, apesar disso, jogadores podem se ajudar para cumprir todas elas.
  • A dificuldade das missões leva como base o jogador de nível mais elevado.
  • O progresso nas minas é global e cada jogador pode abrir os baús por eles mesmos. (usando o elevador para alcançar os níveis necessários também satisfaz as missões da mina)
  • A coleção do Museu é global e cada jogador tem suas próprias recompensas.
  • A recompensa por terminar o conjunto do Mural de Recados é aplicada para todos os jogadores.
  • Completar todos os conjuntos de um local no Centro Comunitário ou alguma das remodelações da cidade no Mercado Joja libera a recompensa final para todos os jogadores presentes no mundo. Contudo, no Centro Comunitário, apenas o jogador que completou determinado conjunto consegue pegar a recompensa dada pelo mesmo.
    • Reparar a ponte na praia também é global.
  • As ferramentas de cada jogador devem ser melhoradas individualmente, mas elas podem ser trocadas pelos jogadores livremente dependendo da necessidade. Isso também elimina qualquer problema possível de não ter uma ferramenta disponível durante uma melhoria.
  • Qualquer jogador pode comprar e colocar construções na fazenda, mesmo assim, apenas o host pode melhorar a Casa da Fazenda e as cabanas só podem ser melhoradas pelos seus donos.
  • Todos os jogadores tem o seu próprio conjunto de Frutas-estrela para se obter, mesmo que a maioria delas estará disponível assim que um dos jogadores alcançar os requerimentos.
  • Receitas para cozinhar e produzir são obtidas individualmente, mas, os jogadores podem ajudar uns aos outros.

Dinheiro

  • Dinheiro pode ser compartilhado por todos os jogadores ou separado individualmente. Isso pode ser definido ao criar um novo jogo ou ao usar a ferramenta presente ao lado do livro do divórcio na casa do prefeito Lewis
  • Para cada item colocado no caixote de entregas, o preço é determinado pelas profissões que mais aumentam seu preço de um jogador online.

Experiência

Cada jogador tem seus próprios pontos de experência. Quem se deu ao trabalho de realizar a ação é quem irá ganhar a experiência.

  • Cultivo
    • A pessoa que colher uma plantação leva a experiência, sem importar quem plantou ou quem teve o trabalho de regá-la todos os dias.
    • O jogador que interagir com um animal, leva e experiência.
  • Mineração
    • O jogador que ter a última batida da picareta na rocha leva a experiência. Não é dividido.
    • O jogador que colocar uma bomba ganha a experiências pelas rochas destruídas.
    • Se uma rocha for destruída por um monstro, como o espírito da poeira, todos ganham a experiência por isso, até mesmo quem não está presente nas minas.
  • Coleta
    • O jogador que coletar o item coletável ganha a experiência.
    • O jogador que tiver a última batida do machado quando cortando uma árvore, galho ou tronco ganha a experiência. Não é dividido.
  • Pesca
    • Você ganha a experiência pelo peixe que capturar.
    • O jogador que pegar o item em um covo ganha a experiência, sem importar quem comprou/produziu ou quem colocou a isca.
  • Combate
    • O jogador que matar o monstro ganha a experiência. Não é dividido.

Sorte

  • Jogadores podem ter diferentes níveis de sorte todos os dias.
  • A sorte do dia é a média da sorte de todos os jogadores.

Tempo

De modo geral, o tempo de um dia do jogo só é pausado quando os jogadores estão em um festival ou quando o comando /pause é utilizado.
Ao hospedar um mundo multijogador, o tempo permanecerá igual ao do modo Um Jogador até que o primeiro jogador adicional entre. Isso significa que o host pode simplesmente manter o inventário ou menu aberto enquanto espera pelos outros jogadores entrarem.

Enquanto estiver num mundo multijogador com dois ou mais jogadores o tempo só pausará quando:

  • O dia terminar e um jogador estiver escolhendo uma profissão ou não fechar o relatório de envios.
  • Os jogadores estiverem participando de um festival que inclui um evento principal. (O tempo será avançado para a noite logo após que o festival acabar.)
  • O host pausar o jogo usando o comando /pause. (Veja abaixo)
  • A hora chegar às duas da manhã e um jogador ainda estiver vendo uma cena. (Ele irá desmaiar logo assim que a cena acabar.)
  • Modo tela divida somente: Se todos os jogadores estiverem no inventário, menu ou cena, o tempo irá pausar.

No multijogador, nenhuma outra circunstância irá fazer o tempo parar, avançar ou correr de forma diferente do normal.

Em particular, o tempo passará enquanto:

  • Em qualquer menu, inventário ou diálogo.
  • Um menu de loja estiver aberto. (A troca ainda estará liberada mesmo quando o NPC tiver saído).
  • Em uma cena. (Ver acima. Você poderá ver os outros jogadores e estará invisível para eles. A hora não é exibida na tela.)
  • Jogando um jogo Arcade.
  • Pescando um peixe.

Algumas ações permanecem possíveis mesmo quando o jogo está pausado pelo comando /pause:

  • Jogadores que estiverem em um menu de loja podem comprar e vender itens.
  • Jogadores que estiverem com seu inventário aberto poderão navegar e mover seus itens.

Comandos

Você pode usar comandos no chat, usando uma barra (/) antes do comando.

Os comandos abaixo são disponíveis para todos os jogadores:

comando descrição
h, help Mostra uma lista de comandos disponíveis.
clear Limpa o histórico de mensagens do jogo.
color-list Mostra uma lista de cores disponíveis com o comando /color.
color Define seu chat para aparecer em determinado cor. Para colorir apenas uma linha do chat: [color]text.
credits Mostra uma lista de créditos do jogo (similar a opção "Créditos" do Menu Principal).
list, users, players Mostra uma lista dos nomes dos jogadores que estão atualmente conectados ao mundo.
ping Mostra o ping do host (para cooperadores) ou dos cooperadores (para o host).


Esses comandos só funcionam para o jogador "host".

comando descrição
pause Pausa o jogo para todos na seção.
resume Retoma o jogo para todos na seção após o comando /pause.
kick <player name> Força uma desconexão de determinado jogador.
ban <player> Bane um jogador da fazenda (não apenas seu personagem). Você pode especificar o nome do jogador, ID ou endereço de IP.
unban <player> Remove o banimento de determinado jogador, permitindo que ele volte a entrar no server.

Bugs

Se você for numa mesa e colocar alguma coisa, e se os jogadores pegarem ao mesmo tempo, se for 2 jogadores duplica o item, se forem 3 jogadores, triplica. Dá para fazer com tudo. Até peixes lendários!
