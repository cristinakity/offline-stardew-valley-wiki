Jogos salvos móveis

  • Língua
  • Vigiar
  • Editar
Para locais dos arquivos de jogos salvos, métodos de salvamento no PC e consoles e edição do arquivo de salvamento, ver Jogos salvos.

Em dispositivos mobile, o método de salvamento de Stardew Valley é baseado em cima do sistema usado para PC, mas com algumas adições para torna-lo mais adequado aos dispositivos mobile.

O jogo cria uma pasta no dispositivo para cada jogo salvo. Você pode ver essa pasta ao conectar seu dispositivo iOS a um Mac ou PC executando o iTunes, assim como destacado aqui.

Nesta pasta existem três tipos de jogos salvos diferentes, os quais podem aparecer - Standard Saves, Emergency Saves e Backup saves.


Standard Saves

Standard Saves são do mesmo formato encontrado na versão do jogo para PC. A versão para PC somente permite o salvamente quando for dormir, assim os stantard saves mantem esse sistema. Um standard save sempre inicia as 6 da manha quando carregado. Esses jogos salvos são compatíveis com a versão de PC e podem ser transferidos de um para o outro.

Quando um standard save acontece, a versão mobile (assim como PC) mantem seus standard saves anteriores, usando o sufixo '_old'. Ocasionalmente os jogos salvos podem encontrar problemas. Se isso acontecer os jogadores podem trocar entra o standard save mais recente e o backup anterior ao pressionar o botão 'Swap to previous save' (Trocar para o save anterior) na página Opções.


Emergency Saves

Diferente de um PC, um dispositivo mobile pode as vezes interromper a execução do jogo - por exemplo - se uma chamada telefônica acontece no meio do jogo. Além disso, os jogadores podem alternar para outro aplicativo, e querer continuar o jogo de onde pararam noutro momento mais tarde. Quando o aplicativo não está mais sendo exibido na tela, é chamado de aplicativo em 'segundo plano'. Se o telefone ou tablet está funcionando com poucos recursos sobrando, pode acabar automaticamente descarregando os aplicativos em segundo plano, assim perdendo qualquer progresso não salvo.

Se o jogo for movido para segundo plano e então resumido, se o aplicativo não tiver sido descarregado pelo telefone ou tablet, o jogo irá ser resumido onde você estava na ultima vez. Para permitir isso, o jogo realiza um 'emergency save' (salvamento de emergência) quando é colocado em segundo plano (por exemplo, quando você pressiona o botão home em um iPhone). Devido à complexidade de Stardew Valley, esse salvamento leva alguns segundos. Se você 'matar' o aplicativo antes desse salvamento ter sido completo (ao deslizar para o lado na lista de aplicativos em segundo plano), o jogo não irá completar o salvamento de emergência (emergency save).

Nota: O jogo não irá realizar um salvamento de emergência quando sair do jogo, se o tempo no jogo for menor que 6:20 da manhã, é assumido que um standard save acabou de ser realizado.

Quando o jogo é carregado ele irá verificar para checar se há um arquivo chamado EMERGENCY_SAVE presente. Se existir, ele navega no arquivo para encontrar a localização mais recente do salvamento de emergência. Se ele um for encontrado o jogador terá a oportunidade de carrega-lo ao invés de acessar o menu principal. O jogo não exclui o emergency save uma vez que ele for carregado. Em vez disso, o emergency save somente será deletado quando for realizado um standard save ou backup save.

Backup Saves

Em qualquer momento (outros durantes certos festivais, cinemáticas ou outros eventos), os jogadores podem manualmente salvar o jogo com o nobotão 'Save Backup' (Salvar backup) na página de opções. Isso salva o progresso como um arquivo de backup, similar ao emergency save, mas os jogadores não poderão carregar esse arquivo quando iniciar o jogo. Em vez disso, quando o jogo for carregado, se existir um backup save que for mais recente que o ultimo standard save, uma janela irá perguntar se o backup save deve ser carregado em vez do standard save. Quando o jogador realiza um backup save, todos os emergency saves são deletados, já que o backup save é mais recente.

Para obter a compatibilidade com a versão de PC, o jogo precisa manter o standard save as 6 da manhã. Mas a versão de PC não é compatível com o conceito de carregamento/salvamento a qualquer momento e qualquer lugar, então por isso que esses diferentes sistemas são suportados
