Frameworks de pacotes de conteúdo são mods feitos a partir do SMAPI que permitem que você escreva pacotes de conteúdo (content packs) com pouca ou até nenhuma experiência em programação e implementar a funcionalidade para você. Pacotes de conteúdos é uma das duas opções disponíveis para fazer um mod para o Stardew Valley, paralelamente com a criação de mods com C# diretamente para o SMAPI.

Cada framework define o próprio formato para o seu pacote de conteúdo; veja a página do framework para saber como utilizar.

Veja essa lista de softwares para pixel art e edição de texto JSON recomendados. Com softwares como VS Code e Sublime, você pode configurar para que ele verifique a formatação do seu arquivo JSON enquanto você escreve seu mod. Dessa forma, é como se fosse uma integração do site de validação de arquivo JSON. Mais detalhes de como fazer esta configuração aqui (página em inglês).

Frameworks populares

Aqui estão os frameworks utilizados com mais frequência para criar pacotes de conteúdo. (O parêntese ao lado de cada nome se referem a sigla convencional usada no nome da pasta.)

Núcleo (1000+ pacotes de conteúdo)

Mod Funcionalidade
Content Patcher (CP) Adicione/edite/substitua qualquer conteúdo no jogo (isto é, conteúdos que normalmente ficam na pasta Content do seu jogo). Eles suporta arquivos de mapas/imagens/dados, tem funcionalidades como patches parciais e prepend/append, e suporta muitas condições e tokens diferentes (por exemplo, criar texturas sazonais, dados que mudam por hora do dia, etc). Uma grande variedade de mods pode ser criada usando Content Patcher, desde simples substituições de retrato até a criação de NPCs inteiros.

Veja Modificações:Content Patcher para obter um guia de introdução.

Json Assets (JA) Crie tipos de itens específicos, com suporte para funcionalidades específicas para cada tipo de item (por exemplo, especificando artesanato/culinária/receitas ou gostos de presentes para novos itens). Atualmente, suporta a criação de objetos, big craftables, colheitas, árvores frutíferas, chapéus, armas, camisas, calças e botas. O conteúdo e as imagens adicionados pelo Json Assets também podem ser editados através do Content Patcher.

Veja o readme no GitHub (página em inglês) para a documentação do mod.

Popular (400+)

Mod Funcionalidade
Alternative Textures (AT) Permite a variação de textura para objetos, construções e entidades. Isso inclui árvores, animais, animais de estimação, plantas e muito mais.
Producer Framework Mod (PFM) Adicionar/alterar as entradas/saídas/comportamento de qualquer máquina no jogo. Isso inclui máquinas do jogo original (por exemplo, barril), máquinas adicionadas por outros mods e adicionando lógica de máquina a qualquer big craftable.

Comum (100+)

Mod Funcionalidade
Better Artisan Good Icons (BAGI) Adicione ícones personalizados a itens de artesanado como Mel ou Vinho com base no item ou flor usada para fazê-los.
Better Farm Animal Variety (BFAV) Adicionar novos tipos de animais.
Custom Furniture (CF) Adicionar novas mobílias.
Custom Music (CM) Substitua ou adicione faixas de música e efeitos sonoros.
Dynamic Game Assets (DGA) Crie tipos de itens específicos, com suporte para funcionalidades específicas para cada tipo de item (por exemplo, especificando artesanato/culinária/receitas ou gostos de presentes para novos itens). Atualmente, ele suporta a criação de big craftables, botas, receitas de artesanato, colheitas, cercas, receitas de forja, árvores frutíferas, mobílias, chapéus, receitas de máquinas, armas melee, objetos, gostos de presentes, calças, camisas, entradas de lojas, receitas de alfaiataria e substituições de textura.
Farm Type Manager (FTM) Adicione lógica de geração semelhante a uma fazenda a qualquer mapa. Isso inclui a inserção de tocos, detritos, forragem, meteoros, monstros, minérios, etc.
Fashion Sense (FS) Permite acessórios maiores, penteados, chapéus, camisas, mangas e calças, além de suporte de animação e muito mais.
Mail Framework Mod (MFM) Implementação nos correios com funcionalidades como planos de fundo personalizados, uma variedade maior de anexos ao correio e suporte para vários anexos. (Correios mais simples sem a funcionalidade personalizada também podem ser adicionados por meio do Content Patcher.)
Shop Tile Framework (STF) Adicione novas lojas ou edite inventários de lojas existentes com uma ampla variedade de condições.
TMXL Map Toolkit (TMXL) Adicionar/editar mapas e locais no jogo. Ele inclui uma grande variedade de funcionalidades de mapa especializadas, como camadas da colheita, a cor da água, ações de blocos, e scripts feitos em Lua.

Veja documentação adicional do TMXL (página em inglês).

Frameworks especializadas

Eles têm menos de 100 pacotes de conteúdo, mas fornecem funcionalidades úteis para casos específicos.

Mod Funcionalidade
Content Patcher Animations (CP-A) Anime qualquer textura carregada através do Content Patcher, isso inclui texturas que normalmente não podem ser animadas (por exemplo, sprites de itens).
Custom Companions (CC) Permite criar entidades personalizadas que podem ser geradas em mapas ou seguir o jogador por aí. Isso inclui animais no mundo ou NPCs errantes.
Custom Crafting Stations (CCS) Adiciona estações de crafting personalizadas o qual abre um menu com receitas limitadas, sem que estas receitas atrapalhem as receitas no menu do jogo original. Essas estações podem ser big craftables do jog original ou ser implementado através do Json Assets (JA). Essas estações podem ser adicionadas também por tiledata integrado ao mapa.
Custom Critters (CC) Adiciona bichos personalizados ao mundo, como borboletas e abelhas.
Custom Farm Loader (CFL) Uma maneira simplificada de adicionar fazendas personalizadas. Suporta, entre outras coisas, a geração de itens/detritos e a definição de regras de pesca dentro do contexto de suas fazendas.
Custom Obelisks (CO) Adiciona obeliscos personalizados que permitem que você se teletransporte da fazenda para qualquer local, bem como totens.
Custom Ore Nodes (CON) Adicione tipos de rocha/minerais que você pode quebrar para obter itens personalizados ou do próprio jogo.
Custom Resource Clumps (CRC) Add new types of boulder-sized resource clumps you can break to get items, either custom or from the base game.
Custom Winter Star Gifts (CWSG) Change the gifts people give at the Winter Star festival via a set of many add and replace instructions.
Expanded Storage (ES) Adiciona armazenamentos personalizados (baús, armários, etc.) que podem ter a capacidade igual ao do jogo original ou além.
Quest Framework (QF) Permite que modders tenham a capacidade de criar missões personalizadas, definir tipos de missões personalizadas e lidar com missões no back-end.
Train Station (TS) Permite que novas localizações sejam acessadas através da ferrovia, fazendo assim que você não precise aplicar correções em caminhos/desvios no mapa.
Unlockable Bundles (UB) Permite a criação de conjuntos personalizados.
