Mods XNB são um tipo de versão antigo de mods e não são mais recomendados. O SMAPI não pode valida-los para verificar compatibilidade. Use por sua conta e risco.

Um mod XNB substitui alguns dos arquivos.xnb do seu jogo para alterar alguma coisa (diferente de mods mais novos que vão na pasta Mods).

Perguntas frequentes

O que são mods XNB?

Mods XNB funcionam da seguinte forma: eles substituem alguns arquivos do seu jogo (que possuem a extensão de arquivo .xnb). Por exemplo, vamos dizer que você queira modificar o retrato da Abigail; você pode substituir o arquivo Content/Portraits/Abigail.xnb com uma versão modificada. (Veja Editando arquivos XNB para obter mais informações sobre o formato XNB.)

Por que não são recomendados?

Mods XNB possuem diversas limitações, muitas vezes entram em conflito uns com os outros, geralmente quebram quando o jogo é atualizado, são difíceis de instalar e remover, etc.

Dê preferencia a mods feitos com SMAPI e pacotes de conteúdo já que estes possuem diversos benefícios sobre os mods XNB:

Recurso Mods XNB Mods SMAPI Pacotes de conteúdo com SMAPI
Facilidade de instalação ✘ Diferente para cada mod ✓ Simplesmente jogar na pasta Mods
Facilidade de desinstalação ✘ Restaurar manualmente os arquivos originais ✓ Basta remover da pasta Mods
Verificação de atualização ✘ Nenhuma ✓ Aviso automático no console do SMAPI
Verificação de compatibilidade ✘ Nenhuma ✓ Mods incompativeis são desabilitados antes que possam causar algum problema
✓ A lista de compatibilidade é regularmente atualizada com atualizações e soluções alternativas
Compatibilidade com outros mods ✘ Cada arquivo só pode ser substituído uma vez ✓ Praticamente universal ✓ Alta compatibilidade
Suporte para tradução ✘ Difícil de traduzir ✓ API de tradução integrada (Vai depender do pacote de conteúdo)
Compatibilidade com o jogo ✘ A maioria das novas atualizações quebram o mod. ✓ SMAPI reescreve automaticamente muitos mods para manter a compatibilidade
✓ Mods open-source são atualizados mais rápidos.

Eles funcionam no Linux, Mac, Windows ou console?

Mods XNB funcionam de forma igual no Linux, Mac, Windows e Android. Consoles não são suportados.

Eles podem corromper meu save?

Sim, mas é incomum. Certifique-se de fazer um back up do seu save regularmente, e remova qualquer conteúdo personalizado antes de desinstalar um mod XNB. Na maioria dos casos, você pode consertar um save corrompido desfazendo seu último save.

Eles desativam as conquistas do Steam?

Não.

Usando Mods XNB

Procurando mods

Veja a página do guia inicial encontrar mods.

Instalando mods

A instalação é diferente para cada mod; verifique sua descrição mod para detalhes. Mas em geral:

  1. Baixe o mod XNB e descompacte-o em algum lugar (não na pasta do seu jogo!).
  2. Ele deve ter conter alguns arquivos com a extensão .xnb.
  3. Encontre a sua pasta do jogo.
  4. Substitua os arquivos .xnb existentes que estão na pasta Content com os do mod. Certifique-se de alterar os arquivos corretos! Existem diferentes arquivos com o mesmo nome (como Content/Data/hats.xnb e Content/Characters/hats.xnb), e substituir o arquivo errado causará erros ou falhas no jogo.

Desinstalando mods

Ou restaure os arquivos originais que foram substituídos (espero que você tenha anotado quais foram!), ou redifina seus arquivos de jogo.

Solução de problemas

Se você sabe quais são os mods que estão causando problema, desinstale-o.

Se você não tem certeza de qual mod está causando o problema, a opção mais simples é:

  1. Redifinir os seus arquivos de jogo.
  2. Reinstalar cada mod XNB um por um;
  3. Teste após cada instalação para se certificar de que nenhum deles causou problemas.
